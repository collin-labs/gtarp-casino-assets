**Fluxo completo de um upgrade item->item (14 passos):**

```
1.  Jogador abre Upgrade -> NUI UpgradeGame.tsx
2.  fetchNui('casino:upgrade:list') -> server retorna inventario + catalogo
3.  Jogador seleciona item proprio (ex: Pistola Tatica, common, G$100)
4.  Grid de alvos filtra: mostra apenas itens com valor > G$100
5.  Jogador seleciona alvo (ex: Sniper Pesada, rare, G$800)
6.  NUI calcula preview: chance = (100/800) * 0.92 = 11.5%, multiplicador = x8.0
7.  Wheel atualiza: zona verde = 11.5% do circulo, vermelha = 88.5%
8.  Jogador clica "UPGRADE!"
9.  fetchNui('casino:upgrade:play') { myItemId: 42, targetItemId: 17 }
10. Server:
    a. Valida: jogador possui item 42? Item 17 existe? Valor 17 > valor 42?
    b. Calcula chance real server-side (nao confia no client)
    c. Gera server_seed, publica hash SHA256
    d. Calcula HMAC_SHA256(server_seed, client_seed + ':' + nonce)
    e. Extrai numero 0.00-99.99
    f. Se numero < chance -> WIN: remove item 42, adiciona item 17 ao inventario
    g. Se numero >= chance -> LOSE: remove item 42 (destruido)
    h. Salva em casino_upgrade_history + casino_upgrade_audit
    i. Retorna resultado completo
11. NUI recebe resultado, inicia animacao do wheel
12. Ticker gira 3-5 voltas (1080-1800 graus) + angulo final calculado
13. Ticker para na zona verde (WIN) ou vermelha (LOSE)
14. Overlay correspondente: vitoria com confetti ou derrota com shake
```
