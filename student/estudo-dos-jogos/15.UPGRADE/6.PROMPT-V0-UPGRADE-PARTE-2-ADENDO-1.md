# COMPONENTE: UpgradeGame

Jogo Upgrade para cassino premium GTA RP (FiveM). Jogador seleciona item do
inventario (ou aposta GCoin) e tenta fazer upgrade para item de maior valor.
Wheel circular SVG com setores coloridos por chance. Visual Las Vegas AAA.

# RESTRICOES TECNICAS

```
- React 19 + TypeScript + Framer Motion 11
- Container: position absolute, inset 0, z-index 60, borderRadius inherit, overflow hidden
- SOMENTE dark mode (#0A0A0A) — NUNCA branco
- CSS inline via style={{}} — NAO Tailwind
- SVG inline para wheel — NAO Canvas
- DOM para: grids, botoes, overlays, modais
- Fontes: Cinzel (titulos), Inter (body), JetBrains Mono (valores)
- Tudo com clamp() — nunca px fixo
- Bilingue: prop lang="br"|"en", textos via { br, en }
- Touch targets min 44px
- Zero deps novas
```
