- https://ssojet.com/hashing/hmac-sha256-in-javascript-in-browser (crypto.subtle.verify pattern)

**Descobertas concretas que impactam abordagem:**
1. Gold metallic REQUER 5-7 stops no gradient (nao 2) — refs CodePen e ibelick confirmam
2. Motion v12 mudou import para "motion/react" — usar "framer-motion" para retrocompatibilidade FiveM
3. Shine effect metalico: ::after com translateX + gradient transparente->branco->transparente
4. useAnimate retorna Promise: ideal para capsulas sequenciais com await
5. Uint8Array.toHex() e 2025+ — FiveM NUI Chromium 93 pode nao ter, usar fallback Array.from().map()
6. crypto.subtle.importKey('raw') funciona em Chromium 93+ — confirmado para FiveM NUI

**O que TERIA feito errado sem pesquisar:**
1. Gradientes metalicos com 2 stops (flat, sem realismo) em vez de 5-7 stops
2. Import "motion/react" que pode nao funcionar no bundle FiveM — manter "framer-motion"
3. Uint8Array.toHex() sem fallback — crash silencioso no NUI
4. Bordas metalicas com border simples em vez do padding-box/border-box trick
