## 6.1 Tabela de Multiplicadores Adaptada para GCoin

A tabela de multiplicadores do Jogo do Bicho tradicional precisa ser adaptada para o contexto do Blackout Casino (GCoins, house edge controlado, economia RP).

### Modo Basico (Apostas por Animal)

| Modo | Animais | Acerto 1a posicao | Acerto outras posicoes | House Edge Implicito |
|------|:-------:|:-----------------:|:---------------------:|:--------------------:|
| Simple | 1 | 12x | 3x | 6% (vs justo 25x/5x) |
| Double | 2 | 90x | 10x | ~8% |
| Triple | 3 | 650x | 40x | ~7% |
| Quadra | 4 | 3.500x | 180x | ~8% |
| Quina | 5 | 15.000x | 1.200x | ~7% |

**Calculo do House Edge (Simple mode):**
- Probabilidade de acertar na 1a posicao: 1/25 = 4%
- Payout justo para 1/25: 25x
- Payout oferecido: 12x
- House edge na 1a posicao: (25-12)/25 = 52% (alto, mas compensado pelos "outros")
- Probabilidade de acertar em QUALQUER das 5 posicoes: 5/25 = 20%
- Payout misto esperado: (1/25 * 12) + (4/25 * 3) = 0.48 + 0.48 = 0.96
- RTP efetivo: 96% (house edge 4% no Simple)

**Para Double mode (2 animais):**
- Probabilidade ambos nas 2 primeiras posicoes: (2/25) * (1/24) = ~0.33%
- Probabilidade ambos em QUALQUER das 5: C(5,2)/C(25,2) = 10/300 = 3.33%
- RTP misto: ~93% (house edge ~7%)

### Modo Avancado (Apostas por Numero)

| Modalidade | Multiplicador (cabeca) | Multiplicador (1o-5o) | Probabilidade | House Edge |
|------------|:---------------------:|:--------------------:|:-------------:|:----------:|
| Grupo | 18x | 3.6x | 1/25 | ~6% |
| Dezena | 55x | 11x | 1/100 | ~8% |
| Centena | 550x | 110x | 1/1.000 | ~8% |
| Milhar | 3.500x | 700x | 1/10.000 | ~12% |
| Duque Grupo | 18x | - | variavel | ~5% |
| Terno Grupo | 120x | - | variavel | ~8% |

**Nota sobre multiplicadores:** Os valores tradicionais (18x, 60x, 600x, 4.000x) sao da banca fisica que opera com margem minima. No Blackout Casino, reduzimos ligeiramente para garantir sustentabilidade economica do server. Os valores propostos (18x, 55x, 550x, 3.500x) mantem a atratividade enquanto dao ~6-12% de house edge, consistente com RTP 88-94%.

# 7. PROVABLY FAIR - DETALHAMENTO TECNICO
# =============================================

## 7.1 Fluxo Completo

```
1. Jogador abre o Animal Game
2. Server gera server_seed (random 32 bytes hex)
3. Server calcula hash = SHA256(server_seed) e envia ao client
4. Client gera client_seed (random ou custom do jogador)
5. Jogador faz apostas e clica JOGAR
6. Client envia: apostas + client_seed ao server
7. Server calcula: result = HMAC_SHA256(server_seed, client_seed + ":" + nonce)
8. Server extrai 5 milhares do hash (5 premios)
9. Server calcula payouts baseado nas apostas
10. Server envia: resultados + payouts ao client
11. Client anima a revelacao dos 5 premios
12. Apos revelacao, server disponibiliza server_seed original
13. Jogador pode verificar: recalcular HMAC e confirmar resultado
14. Nonce incrementa para proxima rodada
```

## 7.2 Extracao de 5 Resultados de um Hash

```javascript
// server-side (handlers/bicho.js)
const crypto = require('crypto');

function gerarResultados(serverSeed, clientSeed, nonce) {
  const hash = crypto
    .createHmac('sha256', serverSeed)
    .update(`${clientSeed}:${nonce}`)
    .digest('hex');

  const resultados = [];
  for (let i = 0; i < 5; i++) {
    const segmento = hash.substring(i * 8, i * 8 + 8);
    const valor = parseInt(segmento, 16);
    const milhar = valor % 10000;
    const dezena = milhar % 100;
    const centena = milhar % 1000;
    const grupo = Math.ceil(dezena / 4) || 25; // 00 = grupo 25

    resultados.push({ milhar, dezena, centena, grupo });
  }
  return { hash, resultados };
}
```

## 7.3 Modal de Verificacao (Client-Side)

O jogador acessa um modal "Provably Fair" que mostra:
- Hash do server seed (mostrado ANTES do jogo)
- Client seed usado
- Nonce da rodada
- Server seed revelado (mostrado APOS o jogo)
- Botao "VERIFICAR" que recalcula o hash no browser e compara

Sem a pesquisa, teriamos:
- Criado um grid simples de 25 animais com clique e resultado aleatorio, sem modos progressivos (Simple/Double/Triple/Quadra/Quina)
- Usado multiplicadores inventados em vez dos tradicionais (18x, 60x, 600x, 4.000x)
- Ignorado o modo avancado com Dezena/Centena/Milhar, que e a ESSENCIA do Jogo do Bicho real
