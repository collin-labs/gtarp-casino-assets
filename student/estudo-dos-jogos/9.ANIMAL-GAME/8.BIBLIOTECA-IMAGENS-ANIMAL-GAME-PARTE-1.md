# PESQUISA REGRA X0 — GERACAO DE IMAGENS 2026

**Rodada 1:** `GPT Image 1.5 API transparent background PNG 2025 2026`
**Rodada 2:** `gpt-image-1.5 transparent holes bug workaround`
**Rodada 3:** `GPT Image 1.5 game asset generation best practices`

**Fontes encontradas:**
1. https://developers.openai.com/api/docs/guides/image-generation — Docs oficiais API
2. https://openai.com/index/new-chatgpt-images-is-here/ — Lancamento GPT Image 1.5
3. https://community.openai.com/t/gpt-image-1-transparency-remove-background-also-cuts-out-other-white-spots-of-the-image/1273481 — Bug transparencia
4. https://evolink.ai/blog/gpt-image-1-5-guide-features-comparison-access — Guia completo 2026
5. https://learn.microsoft.com/en-us/azure/foundry/openai/how-to/dall-e — Azure docs com params

**Descobertas CONCRETAS:**
1. **gpt-image-1.5** e o modelo mais recente (lancado 16/12/2025). DALL-E 3 sera descontinuado em 12/05/2026
2. **Parametros API:** `background:"transparent"`, `quality:"high"`, `output_format:"png"`, sizes: 1024x1024, 1024x1536, 1536x1024
3. **Bug conhecido:** `background:"transparent"` pode cortar partes brancas do objeto (olhos brancos, barrigas claras). Workaround: gerar com fundo branco puro (#FFFFFF) + usar https://transparify.app para remover fundo
4. **gpt-image-1.5 vs gpt-image-1:** 1.5 e 4x mais rapido, melhor instruction-following, melhor consistencia em edicoes sequenciais
5. **Fallback ChatGPT web:** pode gerar direto no chat com "pure white background" e depois remover fundo externamente

**O que TERIA feito errado sem pesquisar:**
- Usado DALL-E 3 (sera descontinuado em 2 meses)
- Nao documentado o bug de transparencia em animais com partes claras (Coelho branco, Vaca branca — grupo 10 e 25)
- Nao incluido fallback para Transparify quando API falhar

**Modelo recomendado:** gpt-image-1.5
**API:** `background:"transparent", quality:"high", output_format:"png"`
**Fallback:** `background:"#FFFFFF"` + https://transparify.app
**ALERTA ANIMAIS CLAROS:** Grupos 10 (Coelho), 25 (Vaca), 1 (Avestruz) — podem ter bug de transparencia. Usar fallback fundo branco para estes.
