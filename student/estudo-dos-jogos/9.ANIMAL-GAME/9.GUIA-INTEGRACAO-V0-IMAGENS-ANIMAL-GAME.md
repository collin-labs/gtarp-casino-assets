Este documento explica como TODAS as imagens deste jogo foram criadas e onde cada uma esta armazenada. As imagens foram geradas via GPT Image 1.5 seguindo estudo de biblioteca de imagens com 7 varreduras obrigatorias nos documentos do projeto.

1. Analisamos 8 documentos do jogo (Pesquisa, Mega Estudo, Guia, Prompt AI, Adendo DS, Roteiro, Prompt-V0, CSS-Componentes)
2. Executamos 7 varreduras sistematicas para encontrar CADA referencia de imagem

4. Geramos prompts individuais com estrutura OBJECT/STYLE/RENDER/COMPOSITION/TECHNICAL/NEGATIVE

6. As imagens foram salvas nas pastas do projeto seguindo nomenclatura padrao

**REGRA PARA O V0:** Ao gerar componentes React, usar os paths EXATOS listados abaixo. NAO inventar paths. NAO usar placeholders. Cada `<img src=...>` e `backgroundImage: url(...)` deve apontar para os arquivos reais deste guia.

**REGRA DE ICONES (X21):** Este jogo usa icones PNG customizados dourados, NAO Lucide icons. Todos os icones estao em `/assets/ui/bicho/`. Tamanho: width/height com clamp(16px, 2vw, 24px).
