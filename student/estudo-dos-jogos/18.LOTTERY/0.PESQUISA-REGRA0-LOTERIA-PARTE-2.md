## 6.1 O que a pesquisa trouxe de NOVO

1. **Loteria ≠ Keno.** Sem a pesquisa, teríamos feito um clone de Keno (grid 80 números, instantâneo). A pesquisa revelou 3 formatos distintos e justificou o híbrido Mega-Sena + Mega Ball que é muito mais rico.

2. **Globo animado com Matter.js é viável.** O artigo do FreeCodeCamp mostrou implementação funcional de bolas quicando dentro de container esférico usando Matter.js. Sem isso, teríamos feito CSS animation genérica.

3. **NENHUM script FiveM tem globo animado.** Analisamos 8 concorrentes — todos são "compra bilhete → notificação de chat". Diferencial total.

4. **Mega Ball multiplicador é o game changer.** O formato da Evolution Gaming com multiplicador 5x-100x após o sorteio principal adiciona uma camada de emoção que não existiria sem pesquisar.

5. **Provably Fair tem algoritmo específico para loterias.** A pesquisa no provably.com e cloudbet.com trouxe o esquema commit-reveal com HMAC_SHA256 que garante transparência sem comprometer segurança.

6. **20 bolas no globo, não 60.** Otimização crucial para NUI: renderizar 60 bolas com physics é pesado. Usar 20 bolas genéricas giratórias (decorativo) e revelar os 6 números sorteados separadamente.

7. **Container circular no Matter.js precisa de polyline.** Sem pesquisar, teríamos tentado criar um corpo circular impossível e perdido tempo. A solução é polyline com 24-32 segmentos.

8. **Mega-Sena usa 2 tambores separados.** Detalhe que enriquece a ambientação — podemos renderizar 2 globos ou usar como referência para o reveal (primeiro a dezena, depois a unidade, estilo real).
