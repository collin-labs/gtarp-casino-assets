# LISTA PROIBIDO — O QUE NAO FAZER (15 itens)

1. PROIBIDO fundo branco, cinza claro, bege, ou qualquer cor clara em qualquer parte da interface
2. PROIBIDO fontes sans-serif para titulos — usar SOMENTE Cinzel Decorative ou Cinzel com serif
3. PROIBIDO dados 2D (emojis de dado, icones planos, imagens PNG de dados) — DEVE ser CSS 3D preserve-3d com 6 faces reais
4. PROIBIDO bordas retas sem border-radius — minimo 6px em qualquer elemento
5. PROIBIDO box-shadow unica (apenas 1 camada) — SEMPRE multicamada: contact shadow + ambient + glow colorido
6. PROIBIDO icones Lucide React ou FontAwesome para identidade visual do jogo — usar PNGs dourados em /assets/dice/icons/. UNICA excecao: botao fechar [X] de modais
7. PROIBIDO tamanhos fixos em px sem clamp() — TUDO responsivo: font-size, padding, margin, width, height, gap
8. PROIBIDO touch targets menores que 44x44px — TODOS os botoes, cards clicaveis, inputs
9. PROIBIDO Canvas 2D ou WebGL para renderizar os dados — usar CSS 3D DOM puro (mais leve no NUI FiveM)
10. PROIBIDO texto em ingles hardcoded sem versao bilingue — TODOS os textos via objeto { br: "...", en: "..." }
11. PROIBIDO confetti ou particulas multicoloridas — SOMENTE dourado: #FFD700, #D4A843, #B8860B
12. PROIBIDO gradientes fora da paleta Blackout (azul generico, roxo, laranja, pink) — apenas paleta documentada
13. PROIBIDO scroll no container principal do jogo — overflow hidden no container, scroll APENAS em areas especificas (historico)
14. PROIBIDO modal sem backdrop-filter blur — TODOS modais usam glassmorphism escuro com blur 12px
15. PROIBIDO resultado dos dados calculado no client — resultado SEMPRE vem do server, client APENAS anima
