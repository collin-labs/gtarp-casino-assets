# DADO 3D STATES CSS (item 7.21 parcial)

```css
/* .dice-3d--idle (antes do roll, sutil pulse) */
transform: perspective(600px) rotateX(0deg) rotateY(0deg);
box-shadow: 0 8px 20px rgba(0, 0, 0, 0.4);
animation: dice-idle-pulse 3s ease-in-out infinite;

/* .dice-3d--spinning (roll ativo, blur + scale) */
transform: perspective(600px) rotateX(var(--rx)) rotateY(var(--ry));
filter: blur(1px);
transform-style: preserve-3d;
animation: dice-spin 0.6s linear infinite;

/* .dice-3d--landing (resultado definido, bounce spring) */
transform: perspective(600px) rotateX(var(--face-rx)) rotateY(var(--face-ry));
filter: blur(0);
animation: dice-bounce 0.4s cubic-bezier(0.34, 1.56, 0.64, 1);
box-shadow: 0 12px 30px rgba(0, 0, 0, 0.5);

/* .dice-3d--stopped (face final visivel, glow sutil) */
transform: perspective(600px) rotateX(var(--face-rx)) rotateY(var(--face-ry));
box-shadow: 0 6px 15px rgba(0, 0, 0, 0.3), 0 0 10px rgba(212, 168, 67, 0.15);
transition: box-shadow 0.5s ease;
```

```css
@keyframes dice-idle-pulse {
  0%, 100% { box-shadow: 0 8px 20px rgba(0,0,0,0.4); }
  50% { box-shadow: 0 8px 25px rgba(0,0,0,0.5); }
}
@keyframes dice-spin {
  0% { transform: perspective(600px) rotateX(0deg) rotateY(0deg); }
  100% { transform: perspective(600px) rotateX(360deg) rotateY(360deg); }
}
@keyframes dice-bounce {
  0% { transform: perspective(600px) rotateX(var(--face-rx)) rotateY(var(--face-ry)) scale(0.85) translateY(10px); }
  60% { transform: perspective(600px) rotateX(var(--face-rx)) rotateY(var(--face-ry)) scale(1.05) translateY(-5px); }
  100% { transform: perspective(600px) rotateX(var(--face-rx)) rotateY(var(--face-ry)) scale(1) translateY(0); }
}
```

## Resumo

| Estado | transform | filter | shadow | animation |
|--------|-----------|--------|--------|-----------|
| idle | neutral | none | 8px dark | pulse 3s |
| spinning | rotating | blur(1px) | none | spin 0.6s |
| landing | face angles | none | 12px dark | bounce 0.4s |
| stopped | face angles | none | 6px + gold glow | none (transition) |
