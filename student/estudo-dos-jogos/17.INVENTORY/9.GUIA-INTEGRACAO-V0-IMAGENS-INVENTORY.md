# INSTRUCAO PARA O V0

Este documento explica como TODAS as imagens deste jogo foram criadas e onde cada uma esta armazenada. As imagens foram geradas via DALL-E / GPT Image seguindo um estudo de biblioteca de imagens com 7 varreduras obrigatorias nos documentos do projeto.

**COMO FOI FEITO:**
1. Analisamos TODOS os documentos do jogo (Prompt AI, Adendo DS, Roteiro, CSS-Componentes)
2. Executamos 7 varreduras sistematicas para encontrar CADA referencia de imagem
3. Mapeamos 21 imagens: 16 itens do inventario + 5 assets de UI
4. Geramos prompts individuais com estrutura OBJECT/STYLE/RENDER/COMPOSITION/TECHNICAL/NEGATIVE
5. Cada imagem foi gerada via DALL-E com fundo transparente (PNG alpha channel)
6. As imagens foram salvas nas pastas do projeto seguindo nomenclatura padrao

**REGRA PARA O V0:** Ao gerar componentes React, usar os paths EXATOS listados abaixo. NAO inventar paths. NAO usar placeholders. Cada `<img src=...>` e `backgroundImage: url(...)` deve apontar para os arquivos reais deste guia.

## SECAO F - ESTILO VISUAL DAS IMAGENS

Todas as imagens foram criadas seguindo o DNA visual do Blackout Casino:

- **Paleta dominante:** Ouro metalico #D4A843, preto profundo #0A0A0A, verde neon #00E676
- **Iluminacao:** Rim light dourado do topo-esquerda, ambient escuro cinematografico
- **Fundo:** Transparente PNG (alpha channel) para TODOS os itens
- **Qualidade:** AAA premium, renders 3D com reflexos metalicos
- **Raridade visual:**
  - Common: acabamento neutro, reflexos azulados sutis
  - Uncommon: detalhes em roxo #8847FF
  - Rare: detalhes em magenta #D32CE6, mais brilho
  - Epic: detalhes em vermelho #EB4B4B, glow sutil
  - Legendary: DOURADO COMPLETO #FFD700, glow intenso, particulas

**O V0 deve manter esse estilo no layout — bordas e feixos em CSS usam as mesmas cores para consistencia visual entre imagem e interface.**
