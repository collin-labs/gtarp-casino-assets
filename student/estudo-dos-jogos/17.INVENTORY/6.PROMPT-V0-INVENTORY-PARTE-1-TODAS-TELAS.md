Crie um componente React TypeScript chamado "InventoryGame".

O componente e a tela do jogo "Inventario / Inventory" para um cassino premium dentro do GTA RP (FiveM). Visual Las Vegas AAA, dark theme, ouro metalico, verde neon. Sistema de inventario de premios ganhos nas caixas do cassino, com cards por raridade, filtros, sell-back e equip.

RESTRICOES TECNICAS:
- React 19 + TypeScript + Framer Motion 11 (importar de "framer-motion")
- Dark mode APENAS - fundo #0A0A0A, NUNCA branco, NUNCA cinza claro
- CSS inline via style={{}} - NAO usar classes Tailwind para componentes do jogo
- SVGs PROIBIDOS para icones de jogo - usar Lucide React apenas para UI (PackageOpen, Search, X, Check, AlertCircle, ChevronLeft, ChevronDown)
- Fontes: 'Cinzel' para titulos, 'Inter' para corpo/labels/botoes, 'JetBrains Mono' para valores monetarios
- Responsividade: TUDO com clamp() - nunca px fixo para fontes/padding
- Framer Motion para: entrada tela, hover/tap cards/botoes, AnimatePresence mode="popLayout" para exit ao vender, layout prop para reordenar grid
- Canvas 2D: NAO necessario - ZERO Canvas neste jogo
- Grid: CSS Grid nativo com auto-fill - NAO usar lib de grid
- Filtros: client-side useMemo - ZERO fetch extra ao filtrar/ordenar
- Bilingue: prop lang="br"|"en", todos textos via objeto { br: "...", en: "..." }
- Export: export function InventoryGame({ onBack, lang })
- Props: onBack() voltar ao menu, lang idioma
- Cores de raridade: Common=#4B69FF, Uncommon=#8847FF, Rare=#D32CE6, Epic=#EB4B4B, Legendary=#FFD700
- Borda e feixo de CADA card usa a COR DA RARIDADE do item (nao dourado fixo)
- ZERO dependencias novas

ESTE PROMPT COBRE 4 TELAS:
1. GRID DO INVENTARIO (tela principal com cards + filtros + header)
2. MODAL DETALHES DO ITEM (glassmorphism com info + botoes)
3. CONFIRMACAO DE VENDA (modal menor com valor)
4. EMPTY STATE (inventario vazio com CTA)
