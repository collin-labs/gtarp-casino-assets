# SECAO 2 — ATMOSFERAS POR ESTADO (5 Estados)

## 2.1 Estado LOBBY_IDLE (Pot Aberto — Navegando)

O jogador acabou de entrar no Jackpot. Ve o donut vazio ou com poucos depositos. Painel de deposito ativo. Lista de jogadores atualizando em tempo real.

```
Mudancas visuais vs estado base:
- Donut: segmentos proporcionais atualizando conforme depositos chegam
  Novo deposito: segmento aparece com Framer Motion scale(0->1) + opacity(0->1)
- Pot total no centro: number ticker animando de 0 ate valor atual
  Novo deposito: ticker re-anima do valor anterior ao novo (ease-out 0.5s)
- Lista jogadores: novo player card aparece com slideDown + fade-in
  Stagger: 0.04s delay por card
- Painel deposito: input focavel, atalhos ativos, botao DEPOSITAR verde pulsando
  Pulsacao sutil: boxShadow oscila entre 0.3 e 0.5 opacidade (2s infinite)
- Sons: ambient loop baixo (5% volume), deposit_received (cada deposito novo)
- Fundo: vinheta padrao, particulas douradas lentas
```

## 2.2 Estado COUNTDOWN (Timer Ativo — 15 Segundos)

O pot atingiu o minimo de jogadores ou tempo. Timer de 15s antes do spin.

```
Mudancas visuais vs LOBBY_IDLE:
- Donut: timer-ring dourado girando ao redor (border-top animation)
  Moldura dourada: glow intensificado (box-shadow 0.5 -> 0.8 opacidade)
- Centro do donut: timer grande substituindo pot value
  Numeros: AnimatePresence scale(2->1) a cada segundo
  Ultimos 5s: cor muda para #FF4444 com glow vermelho
- Painel deposito: AINDA ativo (jogadores podem depositar durante countdown)
  Botao DEPOSITAR: pulsacao mais rapida (1s infinite vs 2s)
  Label adicional: "ULTIMA CHANCE!" em #FFD700 piscando
- Lista jogadores: cards existentes com borda pulsando (indicando pot ativo)
- Sons: countdown_tick (a cada segundo), countdown_urgent (ultimos 5s)
- Fundo: vinheta se fecha levemente (opacidade 0.5 -> 0.7)
  Particulas douradas acelerando
```

## 2.3 Estado SPINNING (Donut Girando — 8 Segundos)

O donut comeca a girar. Depositos travados. Tensao maxima.

```
Mudancas visuais vs COUNTDOWN:
- Donut: transform rotate(Xdeg) com transition 8s cubic-bezier(0.2, 0.8, 0.3, 1)
  X = angulo do ticket vencedor + (360 * 5) para 5 voltas completas
  Velocidade: rapido no inicio, desacelera gradualmente, para suavemente
- Pointer: glow intensificado (#FFD700 com pulsacao rapida 0.5s)
  Cada vez que um segmento passa pelo pointer: flash breve (100ms)
- Painel deposito: DESABILITADO
  Input: disabled, opacity 0.4, pointer-events none
  Botao DEPOSITAR: disabled, grayscale(0.5)
  Label: "GIRANDO..." em #D4A843 com pulse
- Lista jogadores: cards com opacity 0.7, borda pulsando conforme segmento alinha
  Quando o segmento de um jogador passa pelo pointer: card daquele jogador
  tem flash de borda (border-color 1 -> 0.3 em 200ms)
- Sons: spin_start (inicio), spin_loop (durante 8s), spin_decelerate (ultimos 3s)
  Tick a cada segmento que passa pelo pointer (rapido no inicio, lento no final)
- Fundo: vinheta maxima (0.85), particulas douradas intensas, scan line acelerada
```
