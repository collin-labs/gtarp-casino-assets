# DONUT SPINNER STATES — CSS ISOLADO POR ESTADO

O CSS original cobre o donut base mas sem estados isolados como blocos
independentes. Este adendo corrige item 7.21.

---

## Estado: IDLE (pot aberto, esperando jogadores)

```css
/* .jackpot-donut--idle */
background: conic-gradient(rgba(212, 168, 67, 0.15) 0deg 360deg);
box-shadow: 0 0 15px rgba(0, 0, 0, 0.4), inset 0 0 10px rgba(0, 0, 0, 0.3);
opacity: 0.6;
animation: none;
```

---

## Estado: FILLING (jogadores depositando, fatias aparecendo)

```css
/* .jackpot-donut--filling */
background: conic-gradient(/* segmentos dinamicos via JS */);
box-shadow:
  0 0 15px rgba(0, 0, 0, 0.4),
  inset 0 0 10px rgba(0, 0, 0, 0.3),
  0 0 8px rgba(212, 168, 67, 0.1);
opacity: 1;
transition: background 0.5s ease, box-shadow 0.3s ease;
```

```css
/* Nova fatia entrando (via Framer Motion spring) */
/* Segmento novo: opacity 0 -> 1, scale radial 0.9 -> 1 */
```

---

## Estado: SPINNING (girando apos pot fechar)

```css
/* .jackpot-donut--spinning */
box-shadow:
  0 0 30px rgba(212, 168, 67, 0.2),
  inset 0 0 20px rgba(0, 0, 0, 0.4),
  0 0 60px rgba(212, 168, 67, 0.08);
filter: blur(0.5px);
/* rotate controlado por useAnimate com tween ease: [0.12, 0, 0.08, 1] */
/* glow-ring.png sobreposto com animation: pulse 0.5s ease-in-out infinite */
```

```css
/* @keyframes jackpot-glow-pulse */
@keyframes jackpot-glow-pulse {
  0%, 100% { opacity: 0.3; transform: scale(1); }
  50% { opacity: 0.6; transform: scale(1.05); }
}
```

---

## Estado: STOPPING (desacelerando, quase parando)

```css
/* .jackpot-donut--stopping */
box-shadow:
  0 0 25px rgba(255, 215, 0, 0.3),
  inset 0 0 15px rgba(255, 215, 0, 0.1);
filter: none;
/* Framer Motion: velocidade decrescente, ease-out final */
/* Ponteiro: glow dourado intensifica */
```

```css
/* Ponteiro durante stopping */
/* .jackpot-pointer--stopping */
filter: drop-shadow(0 0 10px rgba(255, 215, 0, 0.8));
```

---

## Estado: WINNER (parado no vencedor)

```css
/* .jackpot-donut--winner */
box-shadow:
  0 0 40px rgba(255, 215, 0, 0.5),
  0 0 80px rgba(255, 215, 0, 0.2),
  inset 0 0 20px rgba(255, 215, 0, 0.3);
```

```css
/* Segmento vencedor: pulse dourado */
@keyframes jackpot-winner-pulse {
  0%, 100% {
    filter: brightness(1) drop-shadow(0 0 8px rgba(255, 215, 0, 0.4));
  }
  50% {
    filter: brightness(1.3) drop-shadow(0 0 16px rgba(255, 215, 0, 0.7));
  }
}
/* animation: jackpot-winner-pulse 1.5s ease-in-out infinite */
```
