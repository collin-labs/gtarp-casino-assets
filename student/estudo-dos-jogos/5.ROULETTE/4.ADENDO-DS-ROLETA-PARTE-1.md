Este Adendo DS complementa o Prompt AI da Roleta (#5). Enquanto o Prompt AI
define TELAS com layout ASCII e specs por componente, este documento define
COMPONENTES ISOLADOS com CSS real copiavel, BEM naming, todos os estados,
e referencias exatas ao Design System (DS Parte 1, 2, 3 e Adendo).

Nomenclatura BEM: `.roleta-[bloco]__[elemento]--[modificador]`

Cada componente tem:
- Referencia ao DS (ex: DS P1 ss4.2)
- CSS com `/* COPIAR EXATAMENTE */`
- Estados: normal, hover, active, disabled, variantes do jogo
- Framer Motion props quando aplicavel
- Responsividade com clamp()

Integrar com: `components/games/roulette/*.tsx`
