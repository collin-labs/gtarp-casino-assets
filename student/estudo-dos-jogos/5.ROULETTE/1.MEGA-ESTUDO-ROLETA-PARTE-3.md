**Total handler: ~455 linhas estimadas (arredondado para ~500 com boilerplate)**

**Fluxo completo de um spin (15 passos):**

```
1.  Client: jogador posiciona apostas na mesa (state React local)
2.  Client: clica GIRAR -> fetchNui('roulette:spin', { bets, mode, clientSeed })
3.  Lua: RegisterNUICallback('roulette:spin') -> TriggerServerEvent('casino:roulette:spin', bets, mode, clientSeed)
4.  Server: valida 7 checks (enabled, bets count, amounts, saldo, cooldown, tipos, posicoes)
5.  Server: debita totalBet do saldo do jogador (MySQL UPDATE)
6.  Server: busca seed ativo do jogador (casino_roulette_seeds WHERE is_active=1)
7.  Server: calcula resultado: HMAC-SHA256(serverSeed, clientSeed + ':' + nonce) -> hex -> parseInt(8 chars) % 37
8.  Server: [Lightning] calcula Lucky Numbers: HMAC-SHA256(serverSeed, clientSeed + ':' + nonce + ':lightning') -> 1-5 numeros + multiplicadores
9.  Server: determina cor: REDS.includes(numero) ? 'red' : numero === 0 ? 'green' : 'black'
10. Server: calcula ganhos para cada aposta vs resultado (verificar tipo, posicao, multiplicador se Lightning)
11. Server: credita winAmount ao saldo (MySQL UPDATE)
12. Server: incrementa nonce (casino_roulette_seeds UPDATE nonce = nonce + 1)
13. Server: INSERT casino_roulette_spins + INSERT casino_roulette_audit
14. Server: retorna resultado via callback
15. Client: recebe resultado -> calcula angulo -> anima roda -> overlay resultado -> atualiza saldo/historico
```
