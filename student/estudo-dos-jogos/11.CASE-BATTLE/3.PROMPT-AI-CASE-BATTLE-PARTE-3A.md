# TELA 4: BATALHA — STRIP FRAMER MOTION 11 (substituir GSAP)

O Prompt AI antigo P2 documenta o RevealStrip com timing GSAP (power3.out).

CORRETA para a strip animation via FM11.

**BattleRevealStrip — container:**

**Strip inner (fila de itens — motion.div):**
```
Framer Motion props:
  initial={{ x: 0 }}
  animate={{ x: targetX }}
  transition={{
    duration: stripDuration,
    ease: [0.15, 0.85, 0.35, 1.0]
  }}
  onAnimationComplete={handleRevealComplete}
  style={{ willChange: 'transform' }}

Onde:
  stripDuration = performanceOk ? 4.0 : 2.5 (segundos)
  targetX = -(itemWidth * targetIndex) + (containerWidth / 2) - (itemWidth / 2)
  targetIndex = posicao do item sorteado na fila de ~40 itens

Easing [0.15, 0.85, 0.35, 1.0] produz:
  - Inicio rapido (0.15 control point baixo)
  - Desaceleracao suave e longa (0.85 → 0.35 → 1.0)
  - Efeito "quase parou no anterior" que e o near-miss do strip
```
