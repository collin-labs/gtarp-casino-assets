## FLUXO DO USUÁRIO

### Fluxo Principal: Entrar numa Batalha

1. Jogador abre o Case Battle na aba PVP do painel
2. Vê lista de batalhas abertas com cards visuais (filtráveis por modo)
3. Clica "ENTRAR" numa batalha que interessa → tooltip confirma o custo
4. Sistema verifica saldo, cobra GCoins, navega para sala de espera
5. Na sala, vê slots de jogadores preenchendo (real-time)
6. Quando lota: countdown 3..2..1..GO! → transição para tela de jogo
7. Reveal strips giram simultaneamente para round 1. Som de tick acelerando→desacelerando
8. Item revelado: glow da raridade, score atualiza com counter roll-up
9. Repete para rounds 2, 3... até o último
10. Resultado: vencedor com celebration, perdedores com dimming
11. Prêmio exibido. Botões: Nova Batalha / Lobby / Verificar Fairness

### Fluxo Alternativo: Criar Batalha

1. Jogador clica "+ CRIAR BATALHA" no lobby
2. Modal de criação abre: seleciona caixas (1-5), modo, formato, privada, cashback
3. Preview do custo total. Clica "CONFIRMAR" → modal de confirmação
4. Sistema cria a batalha, cobra entry fee, navega para sala de espera
5. Jogador aguarda. Pode copiar link (se privada). Pode cancelar (se sozinho).
6. Quando alguém entra → mesmo fluxo do principal a partir do passo 5

### Fluxo Alternativo: Cancelar Batalha

1. Na sala de espera, criador está sozinho
2. Clica "CANCELAR" → modal confirmação "Deseja cancelar? Entry fee será devolvida"
3. Confirma → batalha cancelada, reembolso integral, volta ao lobby
4. Toast: "Batalha cancelada. G$ X devolvidos."
