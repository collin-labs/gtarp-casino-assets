# ═══ TELA 1: CARD SELECTOR (BingoGame — estado idle) ═══

Crie um componente React TypeScript chamado "BingoGame".

O componente é a tela do jogo Bingo (#21) para um cassino premium dentro do GTA RP (FiveM). Visual Las Vegas AAA, dark theme, ouro metálico, verde neon. Cartelas 5×5 com Lucky Numbers, ball machine Canvas 2D, 8 padrões de vitória.

RESTRIÇÕES TÉCNICAS:
- React 18 + TypeScript + Framer Motion 11 (importar de "framer-motion")
- Dark mode APENAS — fundo #0A0A0A, NUNCA branco, NUNCA cinza claro
- CSS inline via style={{}} — NÃO usar classes Tailwind para componentes do jogo
- SVGs PROIBIDOS para ícones de jogo — usar Canvas 2D para ball machine
- DOM + CSS Grid para cartelas 5×5 (NÃO Canvas para grids interativos)
- Fontes: 'Cinzel' para títulos, 'Inter' para corpo, 'JetBrains Mono' para valores monetários
- Responsividade: TUDO com clamp() — nunca px fixo para fontes/padding
- Framer Motion para: entrada da tela, stagger cards, scale-in resultado, hover/tap botões
- Lucide React: apenas Check e Lock
- Export: `export default function BingoGame({ onBack, lang })`
- Props: onBack() para voltar ao menu, lang ("br"|"en") para idioma
