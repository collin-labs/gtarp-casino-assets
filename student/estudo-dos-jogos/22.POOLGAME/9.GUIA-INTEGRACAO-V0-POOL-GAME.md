# SECAO F: ESTILO VISUAL

## Paleta
- Ouro: #D4A843, #FFD700, #8B6914
- Verde felt: #1B5E20 → #2E7D32
- Verde neon: #00E676 (guideline, valores)
- Preto: #0A0A0A, #141414

## Iluminacao
- Rim light dourado top-left (UI assets)
- Bolas: gradiente esferico via Canvas arc + radialGradient
- Felt: CSS gradient sutil (nao textura)

## Particularidade Canvas
A mesa de pool e 100% Canvas 2D com Matter.js gravity=0. Nenhum elemento da mesa e PNG. O visual e gerado em runtime por PoolTable.tsx (~350L). Isso torna a Biblioteca de Imagens focada em UI e shop, nao em assets de gameplay.
