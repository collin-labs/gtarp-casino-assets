**Curva Canvas (dentro do requestAnimationFrame loop):**
```javascript
// A cada frame:
// 1. clearRect(0, 0, w, h)
// 2. Desenhar grid (linhas + labels Y)
// 3. Desenhar curva:

ctx.beginPath();
ctx.moveTo(0, h); // comeca no canto inferior esquerdo

// Para cada ponto da curva (1 ponto por pixel horizontal):
for (let px = 0; px <= currentX; px++) {
  const elapsed = (px / w) * totalElapsed;
  const mult = Math.pow(Math.E, GROWTH_RATE * elapsed);
  const y = h - (Math.log(mult) / Math.log(maxScale)) * h;
  ctx.lineTo(px, Math.max(y, 0));
}

// Gradiente da curva:
const grad = ctx.createLinearGradient(0, h, currentX, currentY);
grad.addColorStop(0, '#00E676');
grad.addColorStop(0.6, '#FFD700');
grad.addColorStop(1, '#FF4444');

ctx.strokeStyle = grad;
ctx.lineWidth = 3;
ctx.lineCap = 'round';
ctx.lineJoin = 'round';
ctx.stroke();

// Preenchimento sob a curva (area sutil):
ctx.lineTo(currentX, h);
ctx.lineTo(0, h);
ctx.closePath();
ctx.fillStyle = 'rgba(0,230,118,0.03)';
ctx.fill();

// Ponto final da curva (circulo pulsante):
ctx.beginPath();
ctx.arc(currentX, currentY, 4, 0, Math.PI * 2);
ctx.fillStyle = '#00E676';
ctx.fill();
// Glow fake (circulo maior, semi-transparente):
ctx.beginPath();
ctx.arc(currentX, currentY, 10, 0, Math.PI * 2);
ctx.fillStyle = 'rgba(0,230,118,0.15)';
ctx.fill();
```

**Tom ascendente (Web Audio API — ativo durante RISING):**
```javascript
// Iniciado quando RISING comeca:
const audioCtx = new AudioContext();
const osc = audioCtx.createOscillator();
const gain = audioCtx.createGain();
osc.type = 'sine';
osc.frequency.value = 200; // inicio grave
gain.gain.value = 0.04; // volume baixo
osc.connect(gain);
gain.connect(audioCtx.destination);
osc.start();

// A cada frame, atualizar frequencia:
const minFreq = 200;
const maxFreq = 2000;
const freq = minFreq + (Math.log(currentMultiplier) / Math.log(100)) * (maxFreq - minFreq);
osc.frequency.setTargetAtTime(Math.min(freq, maxFreq), audioCtx.currentTime, 0.1);

// No CRASH, parar suavemente:
gain.gain.exponentialRampToValueAtTime(0.001, audioCtx.currentTime + 0.15);
setTimeout(() => osc.stop(), 200);
```
