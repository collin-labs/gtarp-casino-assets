## 7.4 Probabilidade Detalhada por Pick

A probabilidade de sobreviver ate a N-esima gema com M minas em 25 tiles:

```
P(sobreviver N picks com M minas) = C(25-M, N) / C(25, N)

Onde C(a,b) = a! / (b! * (a-b)!)
```

**Exemplo trabalhado: 3 minas, revelar 5 gemas**

```
P = C(22, 5) / C(25, 5)
  = (22! / (5! * 17!)) / (25! / (5! * 20!))
  = 26.334 / 53.130
  = 0.4955 = 49.55%

Multiplicador justo = 1 / 0.4955 = 2.018x
Com house edge 1% = 2.018 * 0.99 = 1.998x (arredondado: ~2.00x)
```

**Exemplo trabalhado: 5 minas, revelar 3 gemas**

```
P = C(20, 3) / C(25, 3)
  = 1.140 / 2.300
  = 0.4957 = 49.57%

Multiplicador justo = 1 / 0.4957 = 2.017x
Com house edge 1% = 2.017 * 0.99 = 1.997x (arredondado: ~2.00x)
```

Nota: configuracoes diferentes podem resultar em multiplicadores similares quando a probabilidade de sobrevivencia e parecida. Isso e esperado — o multiplicador SEMPRE compensa a probabilidade, descontado o house edge.

## 8.1 Algoritmo (baseado no Stake.com + BC.Game verificado)

```javascript
// 1. ANTES da rodada: server gera server seed e envia hash ao client
const serverSeed = crypto.randomBytes(32).toString('hex'); // 64 chars
const serverSeedHash = sha256(serverSeed); // enviado ao client ANTES da aposta

// 2. Client define client seed (ou usa default) e nonce incrementa

// 3. Ao clicar BET: server calcula posicoes das minas
function gerarPosicoesMinas(serverSeed, clientSeed, nonce, qtdMinas) {
  const hmac = createHmac('sha256', serverSeed);
  hmac.update(`${clientSeed}:${nonce}`);
  const hashHex = hmac.digest('hex');

  // Fisher-Yates shuffle usando bytes do hash como fonte de aleatoriedade
  const indices = Array.from({length: 25}, (_, i) => i);
  for (let i = 24; i > 0; i--) {
    // Extrair 2 bytes do hash para cada swap
    const bytePos = (24 - i) * 2;
    const hexPair = hashHex.substring(bytePos % 64, (bytePos % 64) + 2);
    const randomValue = parseInt(hexPair, 16);
    const j = randomValue % (i + 1);
    [indices[i], indices[j]] = [indices[j], indices[i]];
  }

  // Primeiras N posicoes = minas
  const minas = new Set(indices.slice(0, qtdMinas));
  return {
    grid: Array.from({length: 25}, (_, i) => minas.has(i) ? 'bomb' : 'gem'),
    serverSeedHash,
    serverSeed, // revelado SOMENTE apos a rodada
  };
}

// 4. Apos rodada: server revela serverSeed
// 5. Client pode verificar: sha256(serverSeed) === serverSeedHash
// 6. Client pode recalcular grid com mesmos inputs e confirmar posicoes
```
