# O QUE NAO DEVE TER (LISTA PROIBIDA)

| # | Proibicao | Motivo |
|---|-----------|--------|
| 1 | Fundo branco ou cinza claro em QUALQUER elemento | Dark mode obrigatorio. Fundo mais claro permitido: #2A2A2A (tooltip). |
| 2 | Cores fora da paleta Blackout Casino | Todas as cores devem ser da paleta documentada. Nada de azul, rosa, laranja puro. |
| 3 | Fontes fora do trio Cinzel/Inter/JetBrains Mono | Nenhuma fonte generica (Arial, Helvetica, Times). |
| 4 | px fixo para fontes, padding, gap, width, height | TUDO com clamp(min, vw, max). Unica excecao: border-width (1-2px). |
| 5 | Tailwind classes em componentes do jogo | CSS inline via style={{}}. Tailwind so no painel principal. |
| 6 | Canvas 2D ou WebGL para o grid | Grid e puro CSS Grid + DOM. Nao precisa de Canvas. |
| 7 | Grid maior que 5x5 | Padrao universal e 5x5. Feature futura talvez, v1 nao. |
| 8 | Client calculando posicoes de minas | Resultado 100% server-side. Client so anima. |
| 9 | Grid completo enviado ao client durante rodada | Tile-por-tile. fullGrid so no bust/cashout. |
| 10 | Transicoes sem Framer Motion | TODA animacao usa FM. Nao usar CSS-only para entrada/saida. |
| 11 | Elementos sem tooltip | Todo botao, link, icone interativo tem tooltip bilingue (REGRA X5). |
| 12 | Elementos interativos sem hover/active state | Todo elemento clicavel muda visualmente no hover E no active. |
| 13 | Texto hardcoded so em BR ou so em EN | Bilingue obrigatorio via prop lang + objetos {br, en}. |
| 14 | Sons genericos ou silencio | 14 sons dedicados mapeados. Cada evento tem seu som. |
| 15 | Multiplicador calculado no client sem tabela | Usar tabela pre-calculada do server. Client exibe, nao calcula. |
| 16 | Botao CASH OUT sem animacao pulsante | cashOutGlow keyframe OBRIGATORIO quando ativo. |
| 17 | Resultado sem stagger reveal | Pos-rodada (win ou bust), TODAS tiles revelam em cascata 0.04s/tile. |
| 18 | Modal sem glassmorphism | backdrop-filter: blur(Npx) + background rgba escuro em modais. |
