# SECAO 5 — TIMELINE POR ESTADO DO JOGO

## 5.1 Timeline IDLE → PLAYING (clicou BET)

```
t=0ms    Botao BET pressionado (whileTap scale 0.97)
t=50ms   fetchNui 'mines:bet' enviado ao server
t=50ms   Input aposta e seletor minas ficam disabled (opacity 0.3)
t=50ms   Botao BET some (opacity 0 → hidden)
t=100ms  Server retorna roundId + serverSeedHash
t=100ms  Grid muda de --idle para --playing (opacity 0.6 → 1.0, 200ms)
t=100ms  Tiles ativam hover (spotlight verde)
t=150ms  Botao CASH OUT aparece mas --disabled (precisa 1+ gemas)
t=150ms  Botoes ALEATORIO e AUTO aparecem (FM opacity 0→1, y 10→0, 200ms)
t=200ms  Multiplicador muda de --idle para --active ("1.00x" em dourado)
t=300ms  Som bet_placed.mp3 (300ms)
TOTAL: ~300ms do click ate grid totalmente ativo
```

```
t=0ms    Click na tile (whileTap scale 0.95)

t=50ms   fetchNui 'mines:revealTile' enviado
t=100ms  Server retorna { type: 'gem', multiplier, profits }
t=100ms  Tile --loading → --gem (FM scale [0, 1.2, 1], 400ms)
t=100ms  Som gem_reveal.mp3 (400ms)
t=100ms  Icone gema aparece com drop-shadow

TOTAL: ~500ms do click ate feedback completo
```
