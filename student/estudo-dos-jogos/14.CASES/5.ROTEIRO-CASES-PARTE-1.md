## RELATORIO DA PESQUISA (REGRA X0)

**Pesquisas realizadas:**
1. `oxmysql FiveM query_async best practices transaction 2025 2026` → 10 resultados
2. `HMAC SHA256 Node.js crypto createHmac provably fair casino seed nonce implementation 2025` → 10 resultados

**Fontes relevantes:**
- OxMySQL Docs (coxdocs.dev): `exports.oxmysql.query_async` para queries assincronas. Prepared statements com `?` placeholders previnem SQL injection. Suporta transacoes atomicas
- VertexMods (FiveM Database Guide 2026): MariaDB recomendado sobre MySQL 8 (menos reserved keywords). Connection pooling nativo no oxmysql. `innodb_flush_log_at_trx_commit = 2` para performance FiveM
- Stake.com (Provably Fair Implementation): `createHmac('sha256', serverSeed).update(clientSeed + ':' + nonce + ':' + currentRound)`. 4 bytes → float → roll. cursor incrementa se precisar de >8 outcomes. Server seed hash publicado ANTES da aposta
- Node.js Crypto Docs (v25): `crypto.createHmac('sha256', secret).update(message).digest('hex')` retorna string hex. `crypto.randomBytes(32).toString('hex')` para gerar seeds seguros de 64 chars
- BTCGOSU (Provably Fair Guide 2025): Nonce comeca em 0 e incrementa a cada bet. Server seed revelado no end session. SHA-256 do server seed = hash publico

**O que a pesquisa trouxe de novo:**
1. **oxmysql exports syntax** e `exports.oxmysql.query_async(query, params)` — nao `MySQL.Async.*` que e mysql-async
2. **Provably Fair padrao Stake.com** confirma: `createHmac('sha256', serverSeed).update(clientSeed:nonce:0)` → 4 bytes → float 0-1 → roll 0-999999 → mapeia para item
3. **crypto.randomBytes(32)** gera 32 bytes aleatorios seguros (256 bits) — seed de 64 chars hex
4. **Transacoes atomicas** no oxmysql: `exports.oxmysql.transaction_async([queries])` para debitar saldo + registrar abertura sem race condition

**O que teriamos feito sem a pesquisa:**
- Usado `MySQL.Async.fetchAll` (mysql-async syntax) em vez de `exports.oxmysql.query_async`
- Gerado seeds com `Math.random().toString(36)` (inseguro) em vez de `crypto.randomBytes(32)`
- Nao usado transacoes atomicas no debito de saldo (vulneravel a race condition)

## 1.3 Criar arquivo `server/handlers/cases_engine.js`

**Arquivo NOVO — engine Provably Fair isolado (padrao Stake.com confirmado na pesquisa X0):**

```javascript
// ====================================================
// BLACKOUT CASINO — CASES #14 — PROVABLY FAIR ENGINE
// HMAC-SHA256 padrao Stake.com
// serverSeed + clientSeed + nonce → roll 0-999999 → item
// 22/03/2026
// ====================================================

const crypto = require('crypto');

// gerar server seed seguro (32 bytes = 64 chars hex)
function generateServerSeed() {
  return crypto.randomBytes(32).toString('hex');
}

// gerar hash do server seed (SHA-256) — publicado ANTES da aposta
function hashServerSeed(serverSeed) {
  return crypto.createHash('sha256').update(serverSeed).digest('hex');
}

// gerar roll via HMAC-SHA256 (padrao Stake.com)
// retorna float entre 0 e 1 (precisao de 6 casas decimais)
function generateRoll(serverSeed, clientSeed, nonce) {
  const hmac = crypto.createHmac('sha256', serverSeed);
  hmac.update(`${clientSeed}:${nonce}:0`);
  const buffer = hmac.digest();

  // primeiros 4 bytes → inteiro unsigned 32-bit
  const int32 = (buffer[0] << 24 | buffer[1] << 16 | buffer[2] << 8 | buffer[3]) >>> 0;

  // normalizar para 0-999999
  const roll = int32 % 1000000;

  return roll;
}

// mapear roll para item da caixa (baseado em probabilidades cumulativas)
// items = [{ id, probability, ... }] onde sum(probability) = 100
function rollToItem(roll, items) {
  const normalizedRoll = roll / 10000; // 0-99.9999

  let cumulative = 0;
  for (const item of items) {
    cumulative += item.probability;
    if (normalizedRoll < cumulative) {
      return item;
    }
  }

  // fallback: ultimo item (nao deve acontecer com probabilidades corretas)
  return items[items.length - 1];
}

// verificar resultado (client-side verification)
// recebe seeds + nonce, retorna roll e item — permite ao jogador conferir
function verifyResult(serverSeed, clientSeed, nonce, items) {
  const roll = generateRoll(serverSeed, clientSeed, nonce);
  const item = rollToItem(roll, items);
  return { roll, item, serverSeedHash: hashServerSeed(serverSeed) };
}

// gerar strip decorativa (40 itens aleatorios com o vencedor na posicao 30)
// strip e DECORATIVA — o resultado ja foi determinado pelo roll
function generateStripItems(items, winnerItem, stripLength = 40, winnerPosition = 30) {
  const strip = [];

  for (let i = 0; i < stripLength; i++) {
    if (i === winnerPosition) {
      strip.push(winnerItem);
    } else {
      // item aleatorio ponderado por probabilidade (apenas visual)
      const visualRoll = Math.random() * 100;
      let cumulative = 0;
      let chosen = items[0];
      for (const item of items) {
        cumulative += item.probability;
        if (visualRoll < cumulative) {
          chosen = item;
          break;
        }
      }
      strip.push(chosen);
    }
  }

  return strip;
}

module.exports = {
  generateServerSeed,
  hashServerSeed,
  generateRoll,
  rollToItem,
  verifyResult,
  generateStripItems,
};
```
