# D. DECLARACAO X21 — HIERARQUIA DE ICONES (item 6.14)

O jogo Cases utiliza a seguinte hierarquia visual conforme REGRA X21:

| Tipo | Tecnologia | Itens |
|------|-----------|-------|
| **Imagens de itens** | PNG dourado (assets/cases/itens/) | 35 itens de todas as 6 caixas |
| **Imagens de caixas** | PNG dourado (assets/cases/caixas/) | 6 caixas x 3 estados = 18 PNGs |
| **Icons de UI** | PNG dourado (assets/cases/icons/) | 10 icons: som on, som off, PF, historico, fechar, filtro, ordenar, battle, streak, info |
| **Botoes de acao** | CSS styled (DS P2 §8) | ABRIR, VENDER, MANTER, BATTLE — sem PNG, estilo DS |
| **Fallback** | Lucide React | Apenas para icones genericos sem PNG: ChevronLeft (voltar), X (fechar modal) |

**Regra aplicada:** PNGs dourados para tudo que e visualmente identitario do jogo. Lucide apenas como fallback para acoes genericas de UI. CSS para botoes de acao conforme DS.
