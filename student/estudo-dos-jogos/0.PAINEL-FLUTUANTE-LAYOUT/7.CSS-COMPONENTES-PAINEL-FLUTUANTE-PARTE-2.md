# O QUE NAO DEVE TER (PROIBIDO — 15 itens)

```
1. Fundo #000000 flat sem gradientes (usar bg-casino.png + radial-gradient)
2. box-shadow single-layer (sempre 3-5 camadas)
3. border cor solida dura (sempre rgba com opacity)
4. Cinzel sem text-shadow em titulos ativos
5. JetBrains Mono sem color #00E676 em valores monetarios
6. backdrop-filter blur > 8px em mais de 2 elementos simultaneos
7. @keyframes sem ease customizado (nunca 'linear' em UI, exceto scan line)
8. Botao sem 4 estados (idle, hover, active, disabled)
9. Botao sem min-height 44px
10. Elemento interativo sem transition (nunca snap instantaneo)
11. Canvas 2D sem throttle 30fps (nunca 60fps para particulas decorativas)
12. border-radius > 18px em containers (exceto badges circulares)
13. font-size px fixo (sempre clamp())
14. Classe Tailwind em componentes do painel (usar CSS inline ou BEM)
15. Icones Lucide React (usar PNGs dourados — REGRA X21)
```
