Crie um componente React TypeScript chamado "BlackoutCasino".

O componente e o PAINEL FLUTUANTE (shell/moldura) de um cassino premium dentro do GTA RP (FiveM). Visual Las Vegas AAA, dark theme, ouro metalico, verde neon. O painel contem 5 abas (CASSINO, ARCADE, PVP, LOJA, EVENTOS) com 22 jogos distribuidos. Navegacao por Dock na base. Carrossel hero. Cards com 7 camadas de efeitos visuais. Particulas douradas flutuantes. Spotlight que segue o cursor. Scan line HUD.

RESTRICOES TECNICAS:
- React 19 + TypeScript + Framer Motion 11 (importar de "framer-motion")
- Dark mode APENAS — fundo #0A0A0A, NUNCA branco, NUNCA cinza claro
- CSS inline via style={{}} — NAO usar classes Tailwind para componentes do painel
- Fontes: 'Cinzel' para titulos/botoes, 'Inter' para corpo, 'JetBrains Mono' para valores/numeros
- Responsividade: TUDO com clamp() — nunca px fixo para fontes/padding/gap
- Framer Motion para: entrada sequencial (4 stages), hover/tap, layoutId (dock bar), AnimatePresence mode="wait" (troca abas)
- Bilingue: prop lang="br"|"in", todos os textos via objeto { br: "...", en: "..." }
- Touch targets: minimo 44px em TODOS os botoes e elementos interativos
- Icons: USAR imagens PNG douradas do projeto, NAO Lucide React (REGRA X21)
- Export: export default function BlackoutCasino()
- Canvas 2D para GoldParticles (50 max, 30fps throttle, prop paused quando jogo ativo)
- Sons: 10 eventos via useSoundManager hook (panel_open, tab_switch, card_hover, card_click, modal_open, game_launch, game_back, panel_close, deposit_confirm, error_buzz)

ESTE PROMPT COBRE 10 TELAS/ESTADOS:
