## Y0 — RESUMO DA PESQUISA REGRA 0 (Decisoes Estabelecidas)

A Pesquisa Doc 0 (959L, 3 partes, 12 buscas, 40+ URLs) estabeleceu as seguintes decisoes que GOVERNAM este Mega Estudo:

1. **Engine de animacao:** Framer Motion v12.x (nao GSAP). Layout animations e layoutId sao criticos para transicao de abas e card→modal.
2. **CSS no FiveM:** CEF M103 (Chrome 103, jun/2022). Apenas @property eh seguro entre features modernas. Container queries, CSS nesting, :has(), oklch NAO funcionam. Manter clamp() + CSS classico.
3. **Performance budget NUI:** 30fps particulas, <500 DOM nodes, <50MB heap, <200KB JS gzip, max 2-3 backdrop-filter simultaneos.
4. **Moeda interna:** GCoin como ficha do cassino. Jogador compra com dinheiro do servidor, usa nos jogos, saca de volta.
5. **Concorrentes FiveM:** ZERO paineis de cassino com visual premium integrado. Gap de mercado confirmado.
6. **PWA viavel:** Arquitetura dual FiveM + Web com hook useGameAPI. Componentes React identicos.
7. **Efeitos de luxo:** 18 efeitos catalogados, 8 tecnologias viaveis no CEF M103 (@property, conic-gradient, OffscreenCanvas, SVG filters, Web Animations API, mix-blend-mode, perspective 3D, mask-image).
8. **Sound Design:** 10 eventos sonoros mapeados para o painel (panel_open, tab_switch, card_hover, etc).
