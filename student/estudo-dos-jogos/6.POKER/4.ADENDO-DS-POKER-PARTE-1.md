# COMO USAR ESTE DOCUMENTO

Este Adendo DS e o **complemento tecnico do Prompt AI**. O Prompt AI descreve O QUE cada tela deve ter. Este documento descreve COMO implementar cada componente com CSS copiavel.

**Para cada componente:**
1. `/* COPIAR EXATAMENTE */` — CSS pronto para style={{}} ou className
2. BEM naming `.poker-[bloco]__[elemento]--[modificador]`
3. Referencia exata ao DS (ex: DS P1 §4.2)
4. TODOS os estados visuais (normal, hover, active, disabled, variantes)
5. Framer Motion props exatas quando aplicavel

**Convencao de nomes:**
- `.poker-game` — container raiz
- `.poker-card` — carta individual
- `.poker-dealer` — area do dealer
- `.poker-player` — area do jogador
- `.poker-bet` — controles de aposta
- `.poker-result` — painel de resultado
- `.poker-overlay` — overlays e modais
