# O QUE NAO DEVE TER (PROIBIDO)

| # | PROIBIDO | Por que |
|---|----------|---------|
| 1 | Fundo branco ou cinza claro em qualquer tela | Quebra o visual premium dark do Blackout Casino |
| 2 | Classes Tailwind em componentes do jogo | Projeto usa CSS inline para componentes de jogo |
| 3 | position: fixed em qualquer elemento | NUI FiveM nao suporta corretamente |
| 4 | z-index acima de 90 (exceto modais a 70) | Conflita com dock do painel principal |
| 5 | Imagens cortadas (object-fit: cover em itens) | Itens devem ser contain para mostrar completos |
| 6 | Precos sem icone GCoin | Todo valor monetario DEVE ter o icone G$ antes |
| 7 | Botoes menores que 44px de altura | Viola REGRA X4 (touch targets) |
| 8 | Fontes menores que 9px em qualquer elemento | Ilegivel no NUI FiveM |
| 9 | Spinner de loading (circulo girando) | Projeto usa skeleton pulse, nunca spinner |
| 10 | Select nativo do HTML | Usar custom dropdown sempre |
| 11 | Links externos (window.open) | NUI FiveM nao suporta |
| 12 | Scroll horizontal no grid de itens | Grid deve ser responsivo com auto-fill |
| 13 | Toast sobre toast (pilha sem gap) | Toasts em coluna-reversa com gap 8px |
| 14 | Comprar proprio item (sem validacao) | Validar seller !== currentPlayer |
| 15 | backdrop-filter em cards (so em modais) | Causa flickering no CEF FiveM (Issue #3843) |
