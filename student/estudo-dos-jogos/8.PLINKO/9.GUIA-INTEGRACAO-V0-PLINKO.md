# SECAO A — ITENS/OBJETOS DO JOGO

O jogo Plinko NAO tem itens de inventario com imagem PNG. Os elementos visuais do gameplay sao renderizados via Canvas 2D (nao img tags):

| Elemento | Renderizacao | Detalhes |
|----------|:------------:|---------|
| Pegs (pinos) | Canvas 2D API | `ctx.createRadialGradient` com #D4A843 core + glow fade. Raio solido: pegRadius * 0.6 |
| Bola | Canvas 2D API | Circulo #00E676 solido + radialGradient glow. Trail: 15 posicoes alpha decrescente |
| Peg flash | Canvas 2D API | radialGradient #FFD700 com alpha 1→0 em 150ms ao contato |
| Slots base | Canvas 2D API | roundRect colorido por tier (cinza/verde/dourado/vermelho) + texto JetBrains Mono |

**Nenhum drawImage() e usado no Canvas — tudo e formas geometricas e gradientes.**
