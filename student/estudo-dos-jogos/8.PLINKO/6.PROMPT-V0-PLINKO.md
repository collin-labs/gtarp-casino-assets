Crie um componente React TypeScript chamado "PlinkoGame".

O componente e a tela do jogo "Plinko" para um cassino premium dentro do GTA RP (FiveM). Visual Las Vegas AAA, dark theme, ouro metalico, verde neon. Bola cai por piramide de pegs dourados ate aterrisar em slot com multiplicador. 3 niveis de risco (Low/Medium/High), 8-16 rows configuraveis. Provably Fair HMAC-SHA256.

RESTRICOES TECNICAS:
- React 19 + TypeScript + Framer Motion 11 (importar de "framer-motion")
- Dark mode APENAS — fundo #0A0A0A, NUNCA branco, NUNCA cinza claro
- CSS inline via style={{}} — NAO usar classes Tailwind para componentes do jogo
- Fontes: 'Cinzel Decorative'/'Cinzel' para titulos/botoes, 'Inter' para corpo, 'JetBrains Mono' para valores/numeros
- Responsividade: TUDO com clamp() — nunca px fixo para fontes/padding/sizes
- Board de pegs: Canvas 2D com dual-layer (canvas estatico para pegs+slots, canvas animacao para bola+trail)
- Bola: circulo verde #00E676 com trail de 15 posicoes em alpha decrescente, animacao bezier quadratica com gravity easing (t^2)
- Pegs: circulos dourados #D4A843 com radialGradient para glow, flash #FFD700 por 150ms ao contato
- Slots na base: retangulos coloridos por tier (cinza <= 1x, verde 1.1-5x, dourado 5.1-50x, vermelho 51-1000x)
- Framer Motion para: entrada de componentes, hover/tap em todos interativos, AnimatePresence em overlays/modais, spring physics
- AnimatePresence obrigatorio em: overlays (Result, BigWin), modais (PF, History), badges historico, autobet stats
- Bilingue: prop lang="br"|"en", todos os textos via objeto { br: "...", en: "..." }
- Touch targets: minimo 44px em TODOS os botoes e areas clicaveis
- Icones: PNG dourados customizados em /assets/plinko/icons/, NAO usar Lucide React (UNICA excecao: botao [X] close de modais)
- Export: export default function PlinkoGame({ onBack, lang }: { onBack: () => void; lang: 'br' | 'en' })
