# LISTING STATES — CSS ISOLADO POR ESTADO

O CSS original e o P2-adendo cobrem ProductCard com hover e feixo verde,
mas os estados de listing (active/sold/expired/cancelled) nao estavam
isolados como blocos CSS independentes. Este adendo corrige item 7.21.

---

## Estado: ACTIVE (listing ativo, a venda)

```css
/* .marketplace-listing--active */
border: 1.5px solid rgba(212, 168, 67, 0.3);
background: rgba(17, 17, 17, 0.95);
opacity: 1;
cursor: pointer;
transition: all 0.2s ease;
```

```css
/* .marketplace-listing--active:hover */
border-color: rgba(212, 168, 67, 0.6);
box-shadow:
  0 0 15px rgba(212, 168, 67, 0.08),
  0 8px 20px rgba(0, 0, 0, 0.4),
  inset 0 1px 0 rgba(255, 215, 0, 0.06);
transform: translateY(-2px);
```
