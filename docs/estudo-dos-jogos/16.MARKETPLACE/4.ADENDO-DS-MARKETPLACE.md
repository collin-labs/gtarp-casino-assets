# RELATÓRIO DA PESQUISA (REGRA 0)

## O que foi pesquisado
- DMarket, CSFloat, Skinport: card layouts escuros, hover effects, price displays
- Steam Community Market: simplicidade, price history graph, listing structure
- Stake.com VIP: reward cards com glow por tier, glassmorphism modais
- Marketplace UX guides (Rigby, Purrweb, Algolia): filter patterns, skeleton loading, trust signals

## O que a pesquisa trouxe de novo
1. **Cards escuros com hover glow por raridade** é padrão em TODOS os marketplaces premium (DMarket, CSFloat) — não é decoração, é hierarquia visual
2. **Price history como sparkline NO CARD** é feature do CSFloat que poucos copiam — no Blackout vai ficar no modal (performance) mas com badge de trend no card
3. **Skeleton loading com pulse** é obrigatório em 2025-2026 — "Carregando..." texto é UX de 2015
4. **Glassmorphism para modais** é DS do Blackout (P2 §12.1) e combina com o visual das referências premium
