    SEMPRE clamp(min, vw, max). Errado: font-size: 16px.
    Correto: font-size: clamp(11px, 1.4vw, 16px).

    Botoes: 8-10px, Badges: 4px, Wheel SVG: nao tem border-radius (e circulo).

    ambient + glow colorido para elementos ativos.
