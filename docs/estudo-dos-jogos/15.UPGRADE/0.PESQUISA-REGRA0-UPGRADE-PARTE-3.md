## 6.2 Opcao Alternativa: Upgrade com GCoin (sem item)

Alem do upgrade de item -> item, o sistema pode suportar upgrade de SALDO:

- Jogador deposita X GCoins
- Escolhe multiplicador (x1.5, x2, x3, x5, x10, x25, x50)
- Chance = (1 / multiplicador) * (1 - houseEdge)
- Se ganhou: recebe X * multiplicador em GCoins

## 8.1 O que a pesquisa trouxe de NOVO

1. **Upgrade NAO existe em cassinos tradicionais.** Sem pesquisa, teriamos procurado referencia no Stake.com/888 e nao encontrado nada. A pesquisa revelou que o formato vem do ecossistema CS2/Rust skin gambling (CSGORoll, Clash.gg, RustClash) -- um nicho completamente diferente dos cassinos convencionais. Isso redirecionou toda a abordagem.

2. **ZERO concorrentes FiveM tem Upgrader.** Analisamos 8+ scripts de casino FiveM. Nenhum oferece upgrade de itens. O Blackout Casino sera o PRIMEIRO. Diferencial absoluto que nenhuma pesquisa superficial descobriria.

3. **A mecanica e universal e padronizada.** Todas as plataformas seguem o MESMO fluxo: selecionar item -> selecionar alvo -> sistema calcula chance -> wheel gira -> resultado. A formula chance = (valor_meu / valor_alvo) * (1 - edge) e consistente. Nao precisamos inventar -- o mercado ja convergiu para um padrao testado e comprovado.

4. **House edge padrao do mercado: 5-10%.** RustClash divulga 10% (RTP 92.7%). CSGORoll ~8% (calculado via reviews). Clash.gg 5-8%. Nosso 8% esta no sweet spot -- competitivo sem ser predatorio.

5. **SVG + Framer Motion e a stack perfeita.** A pesquisa no motion.dev confirmou: Framer Motion (agora "Motion") suporta SVG circle/path nativo, animate rotate e GPU-accelerated, useMotionValue evita re-renders. ZERO libs extras necessarias.

6. **Provably Fair do Stake.com tem codigo JS real publicado.** A pagina stake.com/provably-fair/implementation tem o CODIGO FONTE do algoritmo: createHmac('sha256', serverSeed), hmac.update(clientSeed + ':' + nonce), 4 bytes por resultado. Podemos implementar IDENTICO.

7. **Clash.gg permite escolher ponto de inicio do wheel.** Essa variacao adiciona percepcao de controle para o jogador (ilusao, pois resultado ja esta determinado). Feature diferenciada que podemos incluir.

8. **Modo GCoin (sem inventario) simplifica MUITO a primeira fase.** O upgrade nao precisa obrigatoriamente de itens fisicos. Pode funcionar como Dice/Crash com multiplicador escolhido. Isso permite implementar o jogo ANTES do sistema de inventario completo estar pronto.
