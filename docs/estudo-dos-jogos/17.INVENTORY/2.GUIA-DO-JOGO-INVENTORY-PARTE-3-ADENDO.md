## Colecionar como engajamento

O ato de colecionar itens de diferentes raridades cria um ciclo de
completionismo. O jogador quer "ter todos" de cada tier. Ver a colecao
crescer (grid preenchendo) ativa o sistema de recompensa por progresso.
Cada item novo que aparece com borda colorida no grid gera um micro-momento
de satisfacao — verde para uncommon, azul para rare, roxo para epic.

## Raridade como status social

No contexto GTA RP, possuir itens Legendary (borda dourada com glow) serve
como demonstracao de status no servidor. Outros jogadores veem o item equipado
e reconhecem o investimento. O sistema de cores (5 tiers) permite identificacao
instantanea: basta olhar a cor da borda para saber se o item vale algo.

## Sell-back como decisao risk/reward

Vender um item a 60% do valor eh uma decisao estrategica. O jogador pondera:
"preciso de GCoin agora ou guardo para usar depois?" Essa tensao entre liquidez
imediata e valor futuro cria engagement de longo prazo. O modal de confirmacao
com slider de quantidade amplifica essa decisao — cada clique no "+" aumenta
o valor mostrado em verde, criando ancoragem visual.

## Equip como personalizacao

Equipar um item muda a aparencia no servidor. A transicao visual (borda cinza
para borda verde com badge "EQUIPADO") cria feedback instantaneo. O toggle
equip/unequip eh satisfatorio por si so — o jogador "sente" a mudanca.
