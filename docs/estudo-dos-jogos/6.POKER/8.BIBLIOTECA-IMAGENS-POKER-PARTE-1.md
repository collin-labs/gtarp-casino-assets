# PESQUISA REGRA X0 — GERACAO DE IMAGENS 2026

**Rodada 1 — GPT Image 1.5 API:**
- Query: "GPT Image 1.5 API transparent background 2025 2026"
- Fontes: developers.openai.com, openai.com/index/new-chatgpt-images, fal.ai/learn
- Descoberta: gpt-image-1.5 e o modelo mais recente. DALL-E 2 e 3 serao descontinuados em 05/12/2026. API suporta background:"transparent", quality:"high", output_format:"png". Ate 4x mais rapido que gpt-image-1.

**Rodada 2 — Bugs de transparencia:**
- Query: "gpt-image-1 transparency remove background cuts white spots"
- Fonte: community.openai.com (thread maio 2025)
- Descoberta: Bug conhecido — areas claras/brancas do objeto podem ser cortadas junto com o fundo transparente. Acontece em ~80% dos casos com objetos brancos (ex: barriga de pinguim). Workaround: gerar com fundo opaco e usar rembg para remover fundo, OU usar prompt especifico "NO white background, keep all body parts opaque".
- Impacto: Nos prompts do Poker, chips e cartas tem areas claras. Incluir ALERT no TECHNICAL de prompts com objetos claros.

**Rodada 3 — Parametros API recomendados:**
- API: `model: "gpt-image-1.5"`, `background: "transparent"`, `quality: "high"`, `output_format: "png"`, `size: "1024x1024"`
- Para cards wide: `size: "1536x1024"`
- Para pedestais: `size: "1024x1536"`
