## 8.1 Algoritmo

HMAC-SHA256 para determinar a posicao da roda (segmento vencedor).

## 8.2 Fluxo Completo (10 passos)

1. **Pre-jogo:** Servidor gera server_seed aleatorio (crypto.randomBytes(32).toString('hex')). Calcula hash do server_seed com SHA-256 e armazena.
2. **Abertura da tela:** Cliente recebe hash do server_seed (nao o seed em si) via fetchNui. Cliente gera ou usa client_seed existente.
3. **Jogador clica GIRAR:** Cliente envia { action: 'spin', client_seed: '...', nonce: N } ao servidor.
4. **Servidor calcula resultado:** combined = HMAC-SHA256(server_seed, client_seed + ':' + nonce). Primeiro 4 bytes do hash convertidos para inteiro. Inteiro % numero_de_segmentos = indice do segmento vencedor.
5. **Servidor determina premio:** Busca gcoin_amount do segmento vencedor na tabela daily_rewards. Verifica streak. Se milestone (7/14/30), adiciona bonus.
6. **Servidor registra claim:** INSERT em daily_claims com todos os dados (seeds, hash, premio). UPDATE daily_streaks (incrementa streak, atualiza last_claim_at).
7. **Servidor retorna resultado:** { segment: N, prize: 200, streak: 12, milestone: null, server_seed_hash: '...' }.
8. **Cliente anima a roda:** Roda gira e para no segmento N. A animacao e VISUAL — o resultado ja foi decidido no passo 4.
9. **Verificacao (modal PF):** Jogador pode abrir modal "Provably Fair" que mostra: server_seed_hash (recebido antes do spin), client_seed, nonce. Apos revelar server_seed (na proxima sessao), jogador pode verificar que SHA-256(server_seed) == hash recebido.
10. **Rotacao de seeds:** Ao revelar o server_seed antigo, o servidor gera novo server_seed para a proxima sessao. O client_seed pode ser alterado pelo jogador a qualquer momento.

## 8.3 Conversao Hash para Resultado

```javascript
// Converter hash HMAC-SHA256 em indice do segmento
const hmac = crypto.createHmac('sha256', serverSeed);
hmac.update(`${clientSeed}:${nonce}`);
const hash = hmac.digest('hex');

// Primeiros 4 bytes (8 caracteres hex) = inteiro
const intValue = parseInt(hash.substring(0, 8), 16);

// Modulo pelo numero de segmentos
const totalWeight = segments.reduce((sum, s) => sum + s.weight, 0);
let target = intValue % totalWeight;

// Weighted selection
let segmentIndex = 0;
for (let i = 0; i < segments.length; i++) {
  target -= segments[i].weight;
  if (target < 0) {
    segmentIndex = i;
    break;
  }
}
// segmentIndex = indice do segmento vencedor
```
