- Bilingue: prop lang="br"|"en", todos textos via objeto { br: "...", en: "..." }
- Touch targets: minimo 44px em TODOS os botoes (minHeight: 44)
- Icones: SOMENTE PNG custom (REGRA X21) — NUNCA Lucide, NUNCA SVG inline
- Export: export default function GiveawaysGame({ onBack, lang })
