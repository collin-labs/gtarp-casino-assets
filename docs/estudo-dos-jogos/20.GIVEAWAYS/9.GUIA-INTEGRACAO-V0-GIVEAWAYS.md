# INSTRUCAO PARA O V0

Este documento explica como TODAS as imagens do Giveaways foram criadas e onde cada uma esta armazenada. As imagens foram geradas via GPT Image / Midjourney seguindo estudo de biblioteca de imagens com 7 varreduras obrigatorias nos documentos do projeto.

**COMO FOI FEITO:**
1. Analisamos 8 documentos do jogo (Pesquisa, Mega Estudo, Guia, Prompt AI, Adendo DS, Roteiro, Prompt-V0, CSS-Componentes)
2. Executamos 7 varreduras sistematicas para encontrar CADA referencia de imagem
3. Mapeamos 13 imagens novas: 3 visuais (ticket, trophy, crown) + 10 icons premium (REGRA X21)
4. Geramos prompts individuais com estrutura OBJECT/STYLE/RENDER/COMPOSITION/TECHNICAL/NEGATIVE
5. Cada imagem foi gerada em 1024x1024 com fundo transparente (PNG alpha real)
6. Todas as imagens foram salvas em `public/assets/giveaways/`

**REGRA PARA O V0:** Ao gerar componentes React, usar os paths EXATOS listados abaixo. NAO inventar paths. NAO usar placeholders. Cada `<img src=...>` deve apontar para os arquivos reais deste guia.

**REGRA DE ICONES (X21):** Este jogo usa icones PNG customizados dourados metalicos, NAO Lucide icons. Todos os icones estao em `/assets/giveaways/`. Tamanho: width/height com clamp(). Hover: `filter: brightness(1.2)` + `transform: scale(1.05)`. Lucide permitido APENAS para: botao fechar [X] de modais.
