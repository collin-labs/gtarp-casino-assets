**Pesquisas realizadas para o Adendo DS:**
1. `case opening UI design premium dark theme CSS glow rarity tier animation 2025 2026` → 10 resultados
2. `CSS box-shadow multiple layers neon glow pulsating keyframes gaming dark theme 2025 2026` → 10 resultados
3. `CSS card flip 3D perspective rotateY backface-visibility preserve-3d animation dark mode 2025 2026` → 10 resultados

**Fontes relevantes:**
- FreeFrontend (66 CSS Glow Effects): shadow layering com multiplos box-shadow comma-separated para falloff realista. @keyframes modulam opacity e spread-radius para efeito "breathing". filter:drop-shadow() respeita alpha channel de PNGs
- TestMuAI (47 Glowing Effects 2026): neon-pulse keyframes `0%,100% { box-shadow: 0 0 5px cor, 0 0 20px cor; } 50% { box-shadow: 0 0 10px cor, 0 0 40px cor, 0 0 60px cor; }` — 3 camadas no pulse para intensidade variavel
- Toolypet (CSS Box Shadow Guide 2026): layered glows com decreasing opacity (0.4→0.3→0.2→0.1) simulam dispersao real de luz. Multi-layer e o padrao 2026 vs single shadow
- UXPilot (Glassmorphism 2026): dark mode glass usa tints escuros + colored rim (navy, violet, gold) em vez de branco puro. Apple Liquid Glass (iOS 26) como referencia
- Desandro (3D Transforms Card Flip): perspective no pai, preserve-3d no container, backface-visibility hidden em ambas faces. rotateY 180deg no back. Hardware-accelerated no compositor thread (60fps)
- W3bits (CSS Flip Animation): transition 0.25s ease-in-out para flip suave. Perspective 1000px para profundidade natural (800-1200px range ideal)
- CSS-Tricks (Neon Text): multi-layer text-shadow com blur crescente (7px→42px→151px) para glow realista. @keyframes flicker com porcentagens irregulares (18%,22%,25%) para efeito organico

**O que a pesquisa trouxe de novo:**
1. **Glow por raridade deve usar 3-4 camadas de box-shadow** com opacity decrescente — nao 1 unica sombra grande. Padrao 2026 para glow realista
2. **@keyframes pulse para raridade Epic/Legendary** precisa de 3 camadas no 50% (nao 1) para intensidade adequada
3. **Card flip com Framer Motion** deve usar perspective no pai (1000px) e backface-visibility hidden em AMBAS faces, nao so no back
4. **Glassmorphism dark mode** usa colored rim (dourado para o Blackout Casino) em vez de branco. backdrop-filter blur 12-20px
5. **filter:drop-shadow()** e melhor que box-shadow para itens com imagem PNG transparente (respeita alpha channel) — usar nos itens na strip

**O que teriamos feito sem a pesquisa:**
- Glow com 1 camada de box-shadow (flat, sem depth)
- Pulse com 2 estados (dim, sem a terceira camada no pico)
- Card flip sem perspective no pai (flat, sem 3D)
- Glassmorphism com rim branco (harsh no dark mode)
- box-shadow em PNGs transparentes (glow quadrado em vez de contornar o objeto)
