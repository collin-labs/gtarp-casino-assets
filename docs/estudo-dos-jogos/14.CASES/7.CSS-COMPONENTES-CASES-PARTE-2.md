# 27. O QUE NAO TER — PROIBIDO (min 10)

1. Fundo claro — dark mode #0A0A0A obrigatorio
2. Fontes genericas (Arial, system-ui) — Cinzel, Inter, JetBrains Mono
3. Tailwind classes — CSS BEM com .cases- prefix
4. Cores sem hex ("dourado") — usar var(--cases-gold-primary) ou #D4A843
5. Tamanhos fixos px — clamp(min, preferred, max) obrigatorio
6. Botoes < 44px — touch targets obrigatorios
7. Componentes sem hover+active — TODOS estados obrigatorios
8. Glow identico para raridades — 5 niveis distintos (Common→Legendary)
9. SVG para icones — PNGs dourados em /assets/cases/icons/ (X21)
10. box-shadow em PNGs transparentes — usar filter:drop-shadow (respeita alpha)
11. Linear transition em strips — easing [0.15, 0, 0.05, 1] obrigatorio
12. Scroll nativo na strip — Framer Motion animate x
13. perspective no card em vez do pai — perspective no container pai (.cases-fastopen)
14. backface-visibility so no back — hidden em AMBAS faces (frente e verso)
15. Misturar CSS vars de outros jogos — usar --cases- prefix exclusivamente
