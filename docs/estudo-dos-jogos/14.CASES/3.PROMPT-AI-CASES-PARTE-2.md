# TELA 3 — CaseOpening (Strip Horizontal Animada)

A tela mais importante do jogo. O jogador ve uma faixa horizontal com itens passando rapidamente, desacelerando ate parar no item vencedor. A emocao vem da desaceleracao: itens raros passam "quase parando", gerando tensao maxima. O resultado ja foi determinado server-side — a animacao e decorativa.

## Estados Tela 3

| Estado | O que muda |
|--------|-----------|
| build-up | Caixa tremendo no centro (0.8s). Som case_shake. Fundo escuro blur |
| animating | Strip se movendo rapido → lento. Pointer fixo no centro. Ticks sonoros. Label pulsando |
| revealing | Strip parou. Item vencedor no centro do pointer. Glow da raridade comeca a expandir. Delay 200ms |
| transitioning | Fade out da strip, transicao para CaseResult via AnimatePresence |

## Sequencia de Animacao (Timeline ms)

```
t=0ms      → Overlay aparece (fade-in 300ms)
t=300ms    → Caixa aparece no centro
t=400ms    → Caixa comeca a tremer (wiggle 800ms). Som: case_shake
t=1200ms   → Caixa desaparece (scale 0.8, opacity 0, 200ms)
t=1400ms   → Strip aparece (fade-in 200ms). Som: strip_start
t=1600ms   → Strip comeca a se mover. Ticks a cada item passando no centro
t=3500ms   → Strip desacelerando visivelmente. Ticks mais espacados
t=5500ms   → Strip desacelera quase parando. Tensao maxima
t=6100ms   → Strip para. Item vencedor no centro. Som: reveal_{tier}
t=6300ms   → Glow da raridade expande (200ms de delay para jogador processar)
t=6800ms   → Transicao para CaseResult (AnimatePresence exit + enter)
```
