# CASE CARD STATES (item 7.21 parcial)

```css
/* .cases-card--locked (nivel insuficiente ou nao disponivel) */
opacity: 0.4;
filter: grayscale(0.6);
cursor: not-allowed;
pointer-events: none;
position: relative;
/* Padlock overlay via ::after */

/* .cases-card--available (pode abrir) */
opacity: 1;
cursor: pointer;
border: 1.5px solid rgba(212, 168, 67, 0.3);
transition: all 0.2s ease;

/* .cases-card--available:hover */
border-color: rgba(212, 168, 67, 0.6);
transform: translateY(-4px);
box-shadow: 0 0 15px rgba(212, 168, 67, 0.2);

/* .cases-card--opening (abertura em progresso) */
border-color: #FFD700;
box-shadow: 0 0 20px rgba(255, 215, 0, 0.3);
animation: case-opening-pulse 0.8s ease-in-out infinite;

/* .cases-card--opened (ja aberta, checkmark) */
opacity: 0.6;
border-color: rgba(0, 230, 118, 0.3);
cursor: default;
```

```css
@keyframes case-opening-pulse {
  0%, 100% { box-shadow: 0 0 20px rgba(255,215,0,0.3); }
  50% { box-shadow: 0 0 35px rgba(255,215,0,0.5); }
}
```

## Resumo

| Estado | opacity | border | extra |

| available | 1 | gold 0.3 | hover y-4px |
| opening | 1 | gold solid | pulse 0.8s |
| opened | 0.6 | green 0.3 | checkmark overlay |
