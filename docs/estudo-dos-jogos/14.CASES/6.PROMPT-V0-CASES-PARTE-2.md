# O QUE NAO TER — PROIBIDO (min 10)

1. Fundo branco ou qualquer cor clara — dark mode #0A0A0A obrigatorio
2. Fonte Arial/Helvetica sem fallback — usar Cinzel, Inter, JetBrains Mono
3. Tailwind classes nos componentes do jogo — CSS inline via style={{}}
4. Cores sem hex explicito ("dourado","verde") — sempre #D4A843, #00E676, #FF3B3B
5. Tamanhos fixos em px — TUDO com clamp(min, preferred, max)
6. Botoes menores que 44px — touch targets obrigatorios
7. Elementos interativos sem tooltip bilingue — todo botao tem title BR+EN
8. Componentes sem Framer Motion — whileHover, whileTap, AnimatePresence obrigatorios
9. Strip calculando resultado client-side — resultado SEMPRE server-side, strip DECORATIVA
10. Math.random() — HMAC-SHA256 Provably Fair
11. Canvas para strip ou itens — DOM + Framer Motion (Canvas so para GoldParticles)
12. Glow identico para todas raridades — 5 niveis distintos (8px→40px)
13. SVG para icones — usar PNGs dourados em /assets/cases/icons/ (X21)
14. Scroll horizontal nativo na strip — Framer Motion animate x
15. Reescrever componentes existentes inteiros — edicoes cirurgicas str_replace
