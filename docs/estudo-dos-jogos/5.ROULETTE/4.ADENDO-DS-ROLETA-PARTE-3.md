# SECAO 2 - ATMOSFERAS POR ESTADO

## 2.1 Estado IDLE / BETTING (Mesa Ativa, Roda Parada)

Atmosfera de concentracao e decisao. Jogador posiciona fichas:

- **Fundo:** bg-casino.png + vinheta `inset 0 0 150px rgba(0,0,0,0.8)`. Radial dourado sutil no topo.
- **Mesa:** felt `#0D5C2A` com textura. Bordas douradas `rgba(212,168,67,0.5)`. Todas celulas interativas (cursor pointer). Hover glow branco `rgba(255,255,255,0.08)`.
- **Roda:** idle com micro-oscilacao `rotate [0, 2, -2, 0]` duracao 6s infinite. Bola parada no topo. Glow sutil `drop-shadow(0 0 20px rgba(212,168,67,0.15))`.
- **Chips:** todos interativos. Selecionado com glow verde `#00E676`. Stagger entrance 0.05s.
- **Botao GIRAR:** verde ativo se ha apostas na mesa. Disabled cinza se nenhuma aposta.
- **LIMPAR/REPETIR:** outline dourado ativo. Disabled se sem apostas/historico.
- **Historico mini:** badges dos ultimos resultados. Ultimo com scale 1.3 + borda dourada.
- **Sons:** nenhum som ambiente. chip_place.mp3 ao posicionar ficha. chip_remove.mp3 ao remover. button_hover.mp3 ao passar sobre botoes.
- **Particulas:** GoldParticles density baixa (~8), speedY -0.1, cor `#D4A843` opacity 0.1. Sutis.

## 2.2 Estado LIGHTNING PHASE (Revelacao Lucky Numbers)

Atmosfera de tensao eletrica. Drama visual maximo:

- **Overlay:** escurecimento `rgba(0,0,0,0.6)` + `blur(2px)`. Mesa visivel mas dimmed.
- **Flash:** `@keyframes lightningFlash` branco 0.4s por numero. Opacity 0->0.3->0->0.15->0.
- **Shake:** `@keyframes lightningShake` translate +-3px 0.3s por numero.
- **Raios:** PNG sobre cada Lucky Number. `drop-shadow(0 0 12px rgba(255,215,0,0.6))`.
- **Lucky Numbers:** badge circulo `border 3px solid #FFD700` + multiplicador `background: linear-gradient(135deg, #FFD700, #D4A843)`. Scale bounce Framer Motion.
- **Mesa:** numeros Lucky com glow `box-shadow: 0 0 12px rgba(255,215,0,0.3)`. Demais numeros `opacity: 0.4; filter: grayscale(50%)`.
- **Sons:** lightning_intro.mp3 (t=0.5s), lightning_zap.mp3 por numero (stagger 0.8s).
- **Particulas:** desligadas durante Lightning (foco total nos raios).

## 2.3 Estado SPINNING (Roda Girando, Bola Orbitando)

Atmosfera de suspense crescente. Controles travados:

- **Roda:** Framer Motion `animate={{ rotate: finalAngle }}`. Easing `[0.15, 0.85, 0.35, 1.0]`. Glow intensificado `drop-shadow(0 0 30px rgba(212,168,67,0.25))`.
- **Bola:** orbita reversa. Fase 1 (0-70%): rapida raio 82%. Fase 2 (70-90%): queda radial. Fase 3 (90-100%): bounce 2px.
- **Mesa:** `pointer-events: none; opacity: 0.7`. Chips visiveis mas nao interativos.
- **Chips:** `opacity: 0.3; filter: grayscale(80%); pointer-events: none`.
- **GIRAR:** `background: #333; color: #666`. Texto "GIRANDO..." com `@keyframes girarPulse`.
- **LIMPAR/REPETIR:** `opacity: 0.3; pointer-events: none`.
- **Sons:** spin_start.mp3 (t=0), wheel_spin.mp3 loop volume decrescente, ball_spin.mp3 loop frequencia decrescente, ball_drop.mp3 (t=70%), ball_stop.mp3 (t=90%), silencio 500ms.
- **Particulas:** density aumenta levemente (12). Mais rapidas.

# SECAO 3 - TIMELINES EXATAS

## 3.1 Timeline Spin Completo (Modo Classico)

```
t=0ms       Jogador clica GIRAR.
            Client envia fetchNui('roulette:spin', { bets, mode, clientSeed }).
            Botao GIRAR -> --spinning ("GIRANDO...").
            Mesa -> --spinning (pointer-events none, opacity 0.7).
            Chips -> --disabled (opacity 0.3, grayscale 80%).

t=50ms      spin_start.mp3 toca (whoosh mecanico 300ms, volume 80%).

t=100ms     Server responde com { numero, cor, winAmount, ... }.
            Client calcula angulo final: EUROPEAN_SEQUENCE.indexOf(numero) * (360/37).
            Roda inicia Framer Motion animate={{ rotate: finalAngle }}.
            wheel_spin.mp3 inicia loop (volume 70%, decresce linearmente).
            ball_spin.mp3 inicia loop (clicks rapidos, frequencia decresce).

t=100ms     Bola inicia orbita reversa (sentido oposto a roda).
a t=3000ms  Fase 1: raio 82%, velocidade decrescente.

t=3000ms    Bola entra Fase 2: raio 82% -> 62% (queda radial).
            ball_drop.mp3 toca (quique descendente 200ms, volume 70%).
            wheel_spin.mp3 volume -> 30%.

t=3800ms    Bola entra Fase 3: bounce sutil +2px, -1px, 0.
            ball_stop.mp3 toca (click seco 50ms, volume 80%).
            wheel_spin.mp3 fade out -> 0%.
            ball_spin.mp3 para.

t=3800ms    SILENCIO DRAMATICO — 500ms sem nenhum som.
a t=4300ms

t=4300ms    Roda parada. Bola no pocket final.
            ResultOverlay aparece (AnimatePresence fade-in 400ms).

t=4300ms    Circulo numero: scale 0.4->1 bounce (500ms).

t=4800ms    Texto "VOCE GANHOU!" ou "VOCE PERDEU" (fade-in + y ou shake).

t=5000ms    Rolling counter inicia (0->valor em 2s).
            Se vitoria: confetti GoldParticles burst.
            Se derrota: sem confetti.

t=7000ms    Counter termina. Saldo flash (@keyframes win ou lose).

t=7200ms    Botoes GIRAR NOVAMENTE + VOLTAR aparecem (fade-in).

t=7200ms+   Jogador decide: GIRAR NOVAMENTE (volta a IDLE) ou VOLTAR (ModeSelect).
```
