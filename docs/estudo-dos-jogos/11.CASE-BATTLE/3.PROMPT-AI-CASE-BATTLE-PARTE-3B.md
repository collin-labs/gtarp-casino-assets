# RESPONSIVIDADE POR TELA (REGRA X4)
# =============================================

**Tela 1 Lobby:**
- Cards grid: `repeat(auto-fill, minmax(clamp(280px, 40vw, 400px), 1fr))`
- Viewport < 500px: 1 coluna, cards full width
- Filtros: flex-wrap wrap se nao couberem, font reduz via clamp
- Botao CRIAR: full width em mobile, posicao acima dos filtros

**Tela 2 Criar:**
- Modal: width `clamp(320px, 85vw, 580px)`. Mobile: 95vw
- Grid cases: 2 colunas mobile (`minmax(80px, 1fr)`) vs 3-4 desktop
- Toggles modo: viewport < 400px = grid 2x2 em vez de row 4
- Entry fee: font-size reduz de clamp max para min

**Tela 3 Espera:**
- Player circles: flex-wrap. 4-way mobile: grid 2x2
- VS: font reduz via clamp. Mobile < 400px: VS hidden, circulos empilham
- Countdown: font-size clamp(36px, 8vw, 96px) — escala suavemente

**Tela 4 Batalha:**
- 1v1: 2 colunas 50/50 — funciona em qualquer viewport
- 4-way: grid 2x2. Viewport < 500px largura: stack 1 coluna com scroll
- Strip: height `clamp(60px, 8vw, 90px)`, item width `clamp(55px, 7vw, 80px)`
- Round bar + score bar: full width, height via clamp
- Itens revelados: container scrollable se lista > viewport

**Tela 5 Resultado:**
- Colunas itens: `repeat(auto-fit, minmax(clamp(200px, 30vw, 300px), 1fr))`
- Mobile: 1 coluna, vencedor em cima, perdedor abaixo
- Winner banner: font-size `clamp(24px, 5vw, 48px)` — escala
- Botoes acao: flex-wrap, full width em mobile

**Tela 6 PF Modal:**
- Modal: width `clamp(300px, 80vw, 520px)`
- Seed fields: full width, word-break break-all para hashes longos
- Botao CALCULAR: full width sempre
