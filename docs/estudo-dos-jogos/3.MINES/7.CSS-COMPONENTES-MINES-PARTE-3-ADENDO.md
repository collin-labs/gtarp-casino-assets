# ODDS BAR STATES (item 7.21 parcial)

Pesquisa X0: Stake "Profit on Next Tile" display, odds bar mostra chance de
sobreviver em tempo real. Gradient muda de verde (seguro) para vermelho (perigoso).

```css
/* .mines-odds-bar (container) */
width: 100%;
height: clamp(6px, 0.8vw, 10px);
border-radius: 4px;
background: rgba(255, 255, 255, 0.05);
overflow: hidden;
position: relative;

/* .mines-odds-bar__fill--safe (>60% chance) */
background: linear-gradient(90deg, #00E676, #66BB6A);
height: 100%;
border-radius: 4px;
transition: width 0.4s ease;

/* .mines-odds-bar__fill--warning (30-60% chance) */
background: linear-gradient(90deg, #FFD700, #FF9800);
height: 100%;
border-radius: 4px;
transition: width 0.4s ease;

/* .mines-odds-bar__fill--danger (<30% chance) */
background: linear-gradient(90deg, #FF9800, #FF4444);
height: 100%;
border-radius: 4px;
transition: width 0.4s ease;
animation: mines-danger-pulse 1s ease-in-out infinite;

/* .mines-odds-bar__fill--bust (0% — game over) */
background: #FF4444;
width: 100%;
opacity: 0.3;

/* .mines-odds-bar--disabled (antes de apostar) */
opacity: 0.2;
```

```css
@keyframes mines-danger-pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.7; }
}
```

## Resumo

| Estado | Chance | Gradient | animation |
|--------|:------:|---------|-----------|
| safe | >60% | green->green | none |
| warning | 30-60% | gold->orange | none |
| danger | <30% | orange->red | pulse 1s |
| bust | 0% | red solid | none (opacity 0.3) |
| disabled | n/a | none | none (opacity 0.2) |
