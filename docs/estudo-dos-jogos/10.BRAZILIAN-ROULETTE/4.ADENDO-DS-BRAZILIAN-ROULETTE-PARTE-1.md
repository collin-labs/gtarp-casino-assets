## 3.1 Animação de Spin da Roda

```
Duração: 4-6s (random entre esses valores para variação)

GSAP: gsap.to(wheelRef, { rotation: `+=${targetDeg}`, duration, ease })
Bolinha: timing oposto — começa rápido, desacelera mais cedo que a roda
Bounce final: 2-3 mini-bounces de ±2° ao assentar no bolso
Som: spin_start → durante o spin tom crescente → reel_stop no assentamento

## 3.2 Revelação de Multiplicador (Momento Cinematográfico)

```

  t=1000ms — Badge multiplicador aparece: scale(0) → scale(1.5) → scale(1) com bounce

  t=1500ms — Partículas douradas explodem do centro (Canvas burst 30 partículas)
  t=2700ms — Counter termina. Valor final com glow pulsante
  t=3500ms — Som de jackpot hit. Texto "INCRÍVEL!" / "AMAZING!" aparece
