# 4. EIXO 2 — TECNOLOGIA DISPONIVEL

## 4.1 Roda da Roleta — Opcoes Analisadas

A roda da roleta europeia tem 37 setores (0-36), alternando vermelho e preto com o zero em verde. A renderizacao desta roda no navegador pode ser feita de 3 formas. Analisei cada opcao em termos de performance, visual e viabilidade no NUI FiveM.

### Opcao A — Canvas 2D Nativo (ESCOLHIDA)

O HTML5 Canvas com a API 2D oferece controle total sobre a renderizacao dos setores via funcao arc(x, y, radius, startAngle, endAngle). Cada setor e desenhado individualmente com fillStyle para a cor e fillText para o numero. A rotacao e feita via save() + translate() + rotate() no contexto, redesenhando o canvas inteiro a cada frame via requestAnimationFrame. O spin usa easing cubic-bezier personalizado: inicio rapido, desaceleracao gradual, parada suave no numero resultado.

Pattern React confirmado pela pesquisa: useRef para referenciar o elemento canvas, useEffect para lifecycle (iniciar animacao no mount, limpar no unmount), useRef separado para rotationRef (evita re-renders durante animacao — critico para 60fps). O projeto Blackout Casino ja usa Canvas 2D em GoldParticles.tsx e Plinko — equipe familiarizada com o pattern.

**Performance NUI FiveM:** Canvas 2D usa GPU via compositing do CEF (Chromium Embedded Framework). Um canvas de 400x400px com 37 setores + texto + sombra roda a 60fps estavel ate em hardware mid-range. O gargalo NAO e a roda — e o blur/backdrop-filter em OUTROS elementos (bug conhecido CEF Issue #3843).

**Vantagens:** Zero dependencias externas. Controle pixel-perfect. Projeto ja tem experiencia. Performance confirmada no NUI. Suporte a retina/hi-DPI via devicePixelRatio.

**Desvantagens:** Requer mais codigo que libs prontas (~150-200 linhas para roda completa). Sem anti-aliasing automatico em arcos (mitigavel com lineWidth subpixel). Texto em angulo requer translate+rotate por setor.

### Opcao B — Lib react-custom-roulette (npm)

Pacote npm (v1.4.1) que renderiza roda com setores customizaveis. Ultimo update: 2022 (3+ anos sem manutencao). 6 projetos dependentes no npm. API simples: passar array de segmentos, chamar spin(). Internamente usa CSS transforms, nao Canvas.

**Rejeitada porque:** Sem manutencao desde 2022. CSS transforms nao dao controle suficiente sobre easing. Nao renderiza 37 setores com numeros legiveis (feita para 6-12 segmentos). Dependencia externa desnecessaria.

### Opcao C — Lib spin-wheel (npm, crazytim)

Pacote npm (v5.0.2) com roda generica e easing customizavel. Mais recente que react-custom-roulette. Suporte a Canvas e SVG. API de callbacks para spin-start, spin-end.

**Rejeitada porque:** Projetada para rodas genericas (fortune wheel, raffle), nao roleta de cassino com 37 setores precisos. Layout de numeros da roleta europeia (ordem especifica: 0,32,15,19,4,21,...) requer configuracao manual complexa. Canvas nativo com 150L de codigo faz o mesmo com controle total.

### Opcao D — SVG com CSS transforms

Criar 37 paths SVG para os setores e animar a rotacao via CSS transform: rotate(). Abordagem declarativa, sem Canvas imperativo.

**Rejeitada porque:** 37 paths SVG individuais = DOM pesado para NUI FiveM. Animacao CSS nao permite easing dinamico (o numero final e calculado pelo servidor, precisa ajustar o angulo de parada em runtime). Performance inferior ao Canvas em animacoes de alta frequencia.

## 4.4 Codigo Conceitual — Canvas 2D Roda Europeia

```typescript
// Ordem dos numeros na roleta europeia (sentido horario a partir do zero)
const EUROPEAN_WHEEL_ORDER = [
  0, 32, 15, 19, 4, 21, 2, 25, 17, 34, 6, 27, 13, 36,
  11, 30, 8, 23, 10, 5, 24, 16, 33, 1, 20, 14, 31, 9,
  22, 18, 29, 7, 28, 12, 35, 3, 26
];

// Cores: 0=verde, vermelhos e pretos alternados conforme padrao europeu
const RED_NUMBERS = new Set([1,3,5,7,9,12,14,16,18,19,21,23,25,27,30,32,34,36]);
const getColor = (n: number): string => n === 0 ? '#00E676' : RED_NUMBERS.has(n) ? '#C62828' : '#1A1A1A';

function drawWheel(ctx: CanvasRenderingContext2D, cx: number, cy: number, radius: number, rotation: number) {
  const arcSize = (Math.PI * 2) / 37;
  ctx.save();
  ctx.translate(cx, cy);
  ctx.rotate(rotation);
  for (let i = 0; i < 37; i++) {
    const angle = i * arcSize;
    const num = EUROPEAN_WHEEL_ORDER[i];
    ctx.fillStyle = getColor(num);
    ctx.beginPath();
    ctx.moveTo(0, 0);
    ctx.arc(0, 0, radius, angle, angle + arcSize);
    ctx.closePath();
    ctx.fill();
    ctx.stroke();
    // Numero dentro do setor
    ctx.save();
    ctx.rotate(angle + arcSize / 2);
    ctx.translate(radius * 0.82, 0);
    ctx.rotate(Math.PI / 2);
    ctx.fillStyle = '#FFFFFF';
    ctx.font = `bold ${radius * 0.06}px Inter`;
    ctx.textAlign = 'center';
    ctx.fillText(String(num), 0, 0);
    ctx.restore();
  }
  ctx.restore();
}
```
