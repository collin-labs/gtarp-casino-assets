# PROBLEMA: CANVAS 2D COM 3 MODOS DE ROLETA

O BR Roulette tem 3 modos (Relampago 37 casas, Classica 37 casas, Mini 15 casas).
Cada modo usa o mesmo Canvas 2D para a roda, mas com configuracoes diferentes.

Pesquisa X0: GitHub pjhjohn/roulette (deceleration -0.001, acceleration 0.002),
DZone roulette tutorial (arc = Math.PI / segmentos, clearRect por frame),
Konva.js wheel (angularFriction 0.2, numWedges configurable).

## Problema 1: Tres instancias de Canvas simultaneas

Se o jogador troca de modo sem destruir o canvas anterior, a memoria acumula.
CEF do FiveM tem limite de memoria mais restrito que Chrome desktop.

**Solucao:** Uma unica instancia de Canvas compartilhada. Ao trocar modo,
limpar o canvas (clearRect), recalcular segmentos (37 ou 15), redesenhar.
Nunca montar 2+ canvas ao mesmo tempo.

## Problema 2: Deceleration diferente por modo

| Modo | Segmentos | Arc (rad) | Deceleration | Spin duration |
|------|:---------:|:---------:|:------------:|:-------------:|
| Relampago | 37 | 0.1698 | -0.0008 | 6-8s |
| Classica | 37 | 0.1698 | -0.0010 | 8-10s |
| Mini | 15 | 0.4189 | -0.0015 | 4-5s (mais rapido) |

Mini gira mais rapido com arcos maiores (24 graus vs 9.7 graus), criando
sensacao de velocidade diferente. Deceleration maior compensa os arcos maiores.

**Solucao:** Config object por modo com arc, deceleration, spinDuration, colors.
Injetar config ao montar. Nao hardcodar valores no Canvas.

## Problema 3: Ball bounce em 15 vs 37 pockets

No Mini (15 casas), os pockets sao 2.4x maiores. A ball tem mais espaco
para bouncing, o que pode parecer "fake" se os parametros nao forem ajustados.

**Solucao:** bounceAmplitude proporcional ao arcSize. Mini usa bounceAmplitude
menor (2-3 bounces) vs Classica (4-6 bounces). Gravity constante em ambos.

## Problema 4: Lightning overlay sobre Canvas

No modo Relampago, numeros selecionados recebem flash azul SOBRE a roda.
Canvas nao suporta overlay HTML nativo.

**Solucao:** Layer CSS posicional absoluta SOBRE o canvas. Div com
`position: absolute; pointer-events: none;` renderiza os flashes com
Framer Motion (flash scale 1.5->1, opacity 0->1->0, azul eletrico).
Canvas renderiza a roda, overlay renderiza os efeitos.
