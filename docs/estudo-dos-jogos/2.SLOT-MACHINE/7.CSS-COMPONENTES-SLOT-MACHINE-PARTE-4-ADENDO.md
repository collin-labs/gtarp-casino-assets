# TUMBLE CELL STATES CSS (item 7.21 parcial)

Pesquisa X0: Sweet Bonanza "symbols explode and disappear, new symbols
cascade down". Gates of Olympus "Tumble Feature — winning symbols removed,
remaining drop to bottom, new ones tumble from above".

```css
/* .slot-cell--exploding (simbolo removido apos win) */
animation: cell-explode 0.2s ease-out forwards;
pointer-events: none;

/* .slot-cell--falling (simbolo existente descendo para preencher gap) */
animation: cell-fall 0.3s cubic-bezier(0.34, 1.56, 0.64, 1);

/* .slot-cell--entering (novo simbolo entrando de cima) */
animation: cell-enter 0.3s cubic-bezier(0.34, 1.56, 0.64, 1);

/* .slot-cell--multiplier-hit (multiplicador orb landing) */
animation: cell-multiplier-pulse 0.5s ease-in-out;
box-shadow: 0 0 15px rgba(255, 215, 0, 0.4);
```

```css
@keyframes cell-explode {
  0% { transform: scale(1); opacity: 1; }
  50% { transform: scale(1.3); opacity: 0.5; }
  100% { transform: scale(0); opacity: 0; }
}
@keyframes cell-fall {
  0% { transform: translateY(calc(var(--fall-distance) * -1)); }
  70% { transform: translateY(4px); }
  100% { transform: translateY(0); }
}
@keyframes cell-enter {
  0% { transform: translateY(-100%); opacity: 0; }
  60% { transform: translateY(6px); opacity: 1; }
  100% { transform: translateY(0); opacity: 1; }
}
@keyframes cell-multiplier-pulse {
  0%, 100% { box-shadow: 0 0 15px rgba(255,215,0,0.4); transform: scale(1); }
  50% { box-shadow: 0 0 30px rgba(255,215,0,0.6); transform: scale(1.1); }
}
```

## Resumo

| Estado | animation | duration | easing |
|--------|-----------|:--------:|--------|
| exploding | scale 1→1.3→0, opacity→0 | 200ms | ease-out |
| falling | translateY gap→bounce→0 | 300ms | spring |
| entering | translateY -100%→bounce→0 | 300ms | spring |
| multiplier-hit | scale pulse + gold glow | 500ms | ease-in-out |
