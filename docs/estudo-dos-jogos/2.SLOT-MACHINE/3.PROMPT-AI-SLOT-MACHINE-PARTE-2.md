**Animacao de spin por coluna (JS/Framer Motion):**
```tsx
// Cada coluna e um container vertical com simbolos empilhados
// Durante spin: simbolos descem com blur, novos entram do topo
// Ao parar: ultimo conjunto fica visivel com bounce sutil

function SpinColumn({ symbols, isSpinning, stopDelay, onStop }) {
  return (
    <motion.div
      style={{
        display: 'flex',
        flexDirection: 'column',
        gap: 'clamp(3px, 0.4vw, 6px)',
        overflow: 'hidden',
      }}
    >
      <AnimatePresence mode="popLayout">
        {symbols.map((sym, row) => (
          <motion.div
            key={`${sym.id}-${row}`}
            layout
            initial={isSpinning ? { y: -60, opacity: 0 } : false}
            animate={{ y: 0, opacity: 1 }}
            exit={{ y: 60, opacity: 0 }}
            transition={{
              type: 'spring',
              stiffness: 300,
              damping: 25,
              delay: isSpinning ? 0 : stopDelay,
            }}
          />
        ))}
      </AnimatePresence>
    </motion.div>
  );
}
```
