**Fluxo completo de um spin VIDEO (18 passos):**

```
1.  Jogador abre Slots → NUI SlotsGame.tsx
2.  Seleciona modo Video → SlotsModeSelect.tsx
3.  VideoSlot.tsx renderiza grid 6x5 idle com simbolos aleatorios
4.  Jogador configura bet (BetControls): valor, ante bet on/off
5.  Jogador clica SPIN (SpinButton.tsx)
6.  fetchNui('casino:slot:spin', { mode:'video', bet:100, anteBet:true, clientSeed:'abc' })
7.  [Lua Client] TriggerServerEvent('casino:slot:spin', source, params)
8.  [JS Server] Valida: mode, bet range, saldo, cooldown
9.  [JS Server] Gera resultado: HMAC-SHA256(serverSeed, clientSeed+':'+nonce) → grid 6x5
10. [JS Server] Avalia wins: scatter pays (8+ iguais), calcula payout
11. [JS Server] Se win: gera tumble (remove wins, gravity, novos simbolos via hash)
12. [JS Server] Repete tumble ate sem wins (max 50 tumbles)
13. [JS Server] Se 4+ scatters no resultado: trigger free spins
14. [JS Server] Calcula totalWin, debita bet, credita win, atualiza jackpot pool
15. [JS Server] Callback com { grid, tumbles[], wins[], multipliers[], totalWin, hashPre }
16. [Lua Client] SendNUIMessage({ type:'SPIN_RESULT', data })
17. [NUI React] Recebe via useNuiEvent, anima: reel scroll → stop → tumble cycles → win overlay
18. [NUI React] Atualiza saldo, historico, free spins counter se aplicavel
```
