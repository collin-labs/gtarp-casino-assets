# PROVABLY FAIR PARA BREAK (item 5.13)

Pool Game e skill-based, mas o break (primeira tacada que dispersa as bolas) e o ball placement apos foul usam RNG. O PF garante que o RNG nao favorece nenhum jogador.

## Fluxo PF do Break

| Passo | Ator | Acao |
|:-----:|------|------|
| 1 | Server | Gera serverSeed (crypto.randomBytes(32).toString('hex')) |
| 2 | Server | Envia hash SHA-256 do serverSeed ao client |
| 3 | Client | Gera clientSeed (browser random) |
| 4 | Server | HMAC-SHA256(serverSeed, clientSeed:nonce:0) |
| 5 | Server | Hash gera 14 floats (7 bolas x 2 coordenadas x/y) para posicionamento |
| 6 | Server | Aplica posicoes na mesa (dentro da area valida do rack) |
| 7 | Client | Pos-partida: serverSeed revelado para verificacao |

## Codigo Handler — Break Placement

```javascript
// server/handlers/pool.js — break ball placement PF
function generateBreakPlacement(serverSeed, clientSeed, nonce) {
  const hmac = require('crypto').createHmac('sha256', serverSeed);
  hmac.update(`${clientSeed}:${nonce}:0`);
  const hash = hmac.digest('hex');

  // 14 floats para 7 bolas (x,y cada)
  const positions = [];
  for (let i = 0; i < 14; i++) {
    const hexSlice = hash.substring(i * 4, i * 4 + 4);
    const value = parseInt(hexSlice, 16) / 65535; // float 0-1
    positions.push(value);
  }

  // mapear para rack triangle area
  const rackCenterX = TABLE_WIDTH * 0.75;
  const rackCenterY = TABLE_HEIGHT * 0.5;
  const rackRadius = BALL_RADIUS * 4;

  const balls = [];
  for (let b = 0; b < 7; b++) {
    const angle = (positions[b * 2] * Math.PI * 2);
    const dist = positions[b * 2 + 1] * rackRadius;
    balls.push({
      id: b + 1,
      x: rackCenterX + Math.cos(angle) * dist,
      y: rackCenterY + Math.sin(angle) * dist
    });
  }

  return { balls, hash: hash.substring(0, 16) };
}
```

## Rematch Handler

```javascript
function requestRematch(source, matchId) {
  const playerId = getPlayerIdentifier(source);
  const match = finishedMatches.get(matchId);
  if (!match) return { error: 'Partida nao encontrada' };

  const opponentId = match.playerA === playerId ? match.playerB : match.playerA;
  const opponentSource = getPlayerSource(opponentId);
  if (!opponentSource) return { error: 'Oponente offline' };

  pendingRematches.set(matchId, {
    requester: playerId,
    opponent: opponentId,
    buyIn: match.buyIn,
    expiresAt: Date.now() + 30000
  });

  emitNet('casino:pool:rematchRequest', opponentSource, {
    matchId: matchId,
    requester: getPlayerName(source),
    buyIn: match.buyIn
  });

  return { status: 'pending', timeout: 30 };
}
```
