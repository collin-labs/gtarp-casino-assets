## 6.4 Anti-Abuse

- **Server-side physics:** Client envia apenas input (angle, power, spin). Server calcula resultado. Impossível manipular posições
- **Timer por turno:** Auto-foul se exceder tempo (impede stalling)
- **Rate limiting:** Máximo 1 shot por turno, validação server-side
- **Limite diário:** 50 partidas/dia por jogador
- **Disconnect handling:** Jogador que desconecta durante partida perde automaticamente (pot vai para oponente)
- **Audit log:** Cada partida registrada: players, mode, tier, shots[], result, winner, rake, timestamp
