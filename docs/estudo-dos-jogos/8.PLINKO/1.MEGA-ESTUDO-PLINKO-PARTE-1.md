## RELATORIO DA PESQUISA — Y0 RESUMO (REGRA 0)

A pesquisa REGRA 0 do Plinko (790L, 2 partes, 22/03/2026) estabeleceu:

**Formato:** Plinko Solo com 3 niveis de risco (Low/Medium/High), 8-16 rows configuraveis, multiplicadores simetricos baseados em distribuicao binomial. Referencia principal: Stake Original (RTP 99%, house edge 1%).

**Tecnologia:** Canvas 2D manual com path pre-calculado pelo Provably Fair. Matter.js DESCARTADO por nao ser deterministico (incompativel com verificacao pos-jogo). Framer Motion para UI. Zero dependencias novas.

**Provably Fair:** HMAC-SHA256 com server seed + client seed + nonce. Cada row = 1 float do HMAC → left/right. Path inteiro pre-calculado antes da animacao comecar.

**Economia:** RTP 99% constante em todas as configs. Low Risk max 16x, Medium max 110x, High Risk max 1000x (16 rows). Bet range 10-50.000 GCoin.

**Concorrencia FiveM:** Apenas 1 script pago (24.60 EUR) inclui Plinko com UI basica. Zero scripts com visual premium. Gap de mercado significativo.

**Sound:** 14 eventos sonoros mapeados com progressive pitch nos bounces e volume layering hierarquico.

**5 telas previstas:** Principal (board+controles), Overlay Resultado, Modal Provably Fair, Modal Historico, Overlay Big Win.
