# 22. PROIBIDO

1. Fundo branco ou cinza claro
2. Fontes padrão navegador (usar Cinzel/Inter/JetBrains Mono)
3. Cores fora da paleta (#D4A843, #FFD700, #00E676, #0A0A0A)
4. SVG para elementos do jogo (usar Canvas para ball machine, DOM para grids)
5. ease genérico (usar cubic-bezier específico)
6. Botão sem tooltip bilíngue
7. Ação importante sem feedback visual
8. Modal sem botão fechar
9. Texto hardcoded em 1 idioma apenas
10. px fixo para layout (usar clamp())
11. Scroll vertical no container principal
12. Reescrever componentes existentes do projeto
13. Canvas para grids 5×5 (usar DOM + CSS Grid)
14. Célula sem estado visual distinto
15. Lucky Number sem badge ×N
