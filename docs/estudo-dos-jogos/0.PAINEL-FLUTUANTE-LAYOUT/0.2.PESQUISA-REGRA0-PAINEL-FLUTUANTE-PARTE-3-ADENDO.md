## A2.1 Tendencias Visuais 2025-2026 para "Luxo Digital"

A pesquisa identificou 3 tendencias aplicaveis ao painel:

**Dark Glassmorphism Evoluido (2026):** Nao eh o blur basico de 2020. A versao 2026 usa camadas translucidas sutis com noise textures, gradient borders, e ambient gradients por tras. Apple lancou "Liquid Glass" no iOS 26/macOS Tahoe em 2025. A chave eh: o vidro precisa de algo para distorcer — gradientes vibrant atras do UI (dourado/verde no caso do Blackout). Opacidade ideal: 10-40%. Blur radius: 10-20px para frosted look sem perder sharpness do texto. No FiveM, limitar backdrop-filter a 2-3 elementos simultaneos (painel + modal) e blur max 12-16px por conta da GPU compartilhada.

**Aurora UI (Gradient Meshes Animados):** Stripe, Linear e Vercel usam blobs com heavy blur movendo suavemente sobre fundo escuro. O componente GoldParticles.tsx do Blackout ja cria efeito similar. Uma evolucao seria adicionar 2-3 blobs dourados/verdes com blur pesado movendo lentamente atras do grid — porem usando PNG pre-renderizado com opacity animada (zero GPU vs 5-10% GPU do blur em tempo real).

**Noise Texture Overlay:** Tendencia 2026 de adicionar ruido sutil sobre superficies para criar sensacao tatil (tipo papel, metal granulado). Implementacao: PNG 256x256 tiled com opacity 3-5% sobre o painel. Custo: ~1% GPU. Efeito: painel parece "fisico", nao "tela de computador".
