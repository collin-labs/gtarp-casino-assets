## VISÃO GERAL

O Jackpot do Blackout Casino é o jogo PvP mais intenso do cassino — uma arena onde 2 a 10 jogadores depositam GCoin num pot compartilhado, cada um aumentando suas chances proporcionalmente ao valor investido, até que uma roda dourada ornamentada gira e seleciona um único vencedor que leva tudo. A tensão começa no primeiro depósito e escala segundo a segundo: cada vez que um novo jogador entra ou alguém aumenta sua aposta, o donut chart se reconfiguRa em tempo real, os percentuais mudam, e a estratégia de todos é forçada a se adaptar. Quando o timer zera, o silêncio dura 8 segundos enquanto a roda gira sob partículas douradas — e quando para, um jogador sai com o pot inteiro menos 5% da casa.

O overlay de vitória explode com confetti dourado, counter de valor em verde neon, e som de fanfarra sincronizado — tudo dentro do painel flutuante do Blackout Casino, herdando bordas, feixos, partículas e tipografia Cinzel.

## FLUXO DO USUÁRIO

### Fluxo Principal: Participar de uma Rodada

1. **Jogador abre o Jackpot** → Clica no card #13 "Jackpot" na aba PVP → Modal "JOGAR AGORA" → Entra no jogo
2. **Vê o estado atual** → Se rodada ativa: donut chart com jogadores atuais + timer + pot total. Se nenhuma rodada: "Aguardando jogadores... seja o primeiro!"
3. **Digita valor** → Preenche input ou usa botão rápido [x2] [MAX] etc.
4. **Clica DEPOSITAR** → Ripple dourado + som coin_deposit + saldo atualiza instantaneamente + donut chart ganha nova fatia com sua cor + lista de jogadores atualiza com sua entrada + timer continua
5. **Observa outros entrarem** → Donut reconfigura em tempo real. Percentuais mudam. Pode depositar novamente para aumentar chance.
6. **Timer nos últimos 10s** → Timer pisca vermelho/dourado. Urgência visual. Se alguém deposita: anti-snipe reseta timer para 10s (flash "ANTI-SNIPE" na UI).
7. **Timer zera** → "RODADA FECHADA" aparece. Donut trava. 1s de pausa dramática.
8. **Spinner gira** → Roda acelera (2s) → gira rápido (3s) → desacelera dramaticamente (3s) → para no setor vencedor. Som de tick desacelera junto. Partículas douradas intensificam.
9. **Resultado** →
   - **Se ganhou:** Flash verde + confetti + "VOCÊ GANHOU! +4.750 GCoin" em counter + som jackpot_win + saldo atualiza
   - **Se perdeu:** Flash vermelho sutil + shake leve + som loss_soft + "Player_X venceu com 32%"
10. **Cooldown 5s** → Resultado some. "Nova rodada em 5... 4... 3..."
11. **Nova rodada abre** → Volta ao passo 2.

### Fluxo Secundário: Verificar Provably Fair

1. **Clica no hash de um round** (no histórico) → Modal ProvablyFair abre
2. **Vê os campos:** Server Seed Hash (publicado antes), Server Seed (revelado após), Client Seeds, Nonce, Winning Ticket
3. **Clica "Verificar"** → Sistema recalcula SHA256(serverSeed) e compara com hash publicado. Se bate: "✅ Round verificado — resultado justo". Se não bate: "❌ Inconsistência detectada".
4. **Fecha modal** → Volta ao jogo
