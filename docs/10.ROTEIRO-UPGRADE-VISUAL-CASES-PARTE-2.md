## 2.2 Helper: Resolver path dos 3 estados da caixa

A função calcula o path baseado no `image_url` que veio do banco (que é o FECHADA).

Adicionar no TOPO do CaseCard.tsx, após os imports:

```tsx
// Helper: gerar paths dos 3 estados a partir do image_url FECHADA
function getCaseStates(imageUrl: string | null) {
  if (!imageUrl) return { closed: null, semi: null, open: null };
  return {
    closed: imageUrl,
    semi: imageUrl.replace("-FECHADA", "-SEMI-ABERTA"),
    open: imageUrl.replace("-FECHADA", "-ABERTA"),
  };
}
```
